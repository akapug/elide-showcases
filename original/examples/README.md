# Original Examples

3 educational projects demonstrating Elide features.

## Projects

### modern-typescript/
Advanced TypeScript patterns and features:
- Generics, conditional types, mapped types
- Decorators and reflection
- Advanced type inference

### real-world/
Production-ready API example:
- RESTful API design
- Error handling
- Validation and middleware

### advanced-typescript/
Advanced TypeScript showcase (from categories/advanced):
- Type system features
- Compiler API usage
- Advanced patterns

## Purpose

These examples are designed for learning, not production use.
They demonstrate what's possible with Elide's TypeScript support.
