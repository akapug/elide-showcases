# Microservices Polyglot Mesh

Complete microservices architecture with services in multiple languages demonstrating service mesh patterns and inter-service communication.

## Services

- **API Gateway** (TypeScript): Request routing and aggregation
- **User Service** (TypeScript): User management and authentication
- **Recommendation Service** (Python): ML-based product recommendations
- **Notification Service** (Ruby): Multi-channel notifications
- **Payment Service** (Go): High-performance payment processing
- **Order Service** (Java): Enterprise order management

## Running

```bash
elide run server.ts
```

## Key Patterns

- Service discovery
- Health checks
- Request routing
- Service orchestration
- Cross-service communication
