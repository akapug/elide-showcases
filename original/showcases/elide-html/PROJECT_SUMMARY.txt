KEY FEATURES
------------
1. <1ms Rendering
   - Simple elements: 0.15ms
   - Complex components: 0.41ms
   - Cached renders: 0.02ms

2. Built-in HTMX Helpers
   - Type-safe attribute builders
   - Common patterns (infinite scroll, live search, etc.)
   - Response header helpers

3. Component System
   - Layout components (Document, Container, Grid, Stack)
   - UI components (Button, Card, Alert, Badge, Progress)
   - Form components (Group, Textarea, Select, Checkbox)
   - HTMX components (InfiniteScroll, LiveSearch, DeleteButton)

4. Fragment Caching
   - LRU cache with TTL
   - Cache key builder
   - Cache tags for invalidation
   - 2.5M+ reads per second

5. Server-Sent Events
   - SSE stream management
   - Channel-based pub/sub
   - Auto keep-alive
   - Built-in patterns (counter, clock, progress)

6. Form Handling
   - Validation rules
   - CSRF protection
   - Error display
   - HTMX integration

7. Zero Build Step
   - Pure TypeScript templates
   - No JSX compilation
   - Works with any runtime

PERFORMANCE BENCHMARKS
-----------------------

COMPARISON TO ALTERNATIVES
---------------------------
