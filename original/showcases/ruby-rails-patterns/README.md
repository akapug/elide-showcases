# Ruby Rails Patterns

**Ruby Rails-style ORM + TypeScript**

Direct Ruby ActiveRecord pattern integration from TypeScript!

## Features
- ActiveRecord pattern in Ruby
- Direct imports in TypeScript
- Model relationships
- Query builders

## Quick Start
```bash
elide run server.ts
```

## API
- `GET /api/users` - List users
- `POST /api/users` - Create user
- `POST /api/seed` - Seed data
