# Backend-for-Frontend (BFF) Polyglot

Client-specific backends optimized for different frontend platforms.

## BFF Implementations

- **Web BFF** (TypeScript): Rich data for web browsers
- **Mobile BFF** (Swift/Kotlin): Minimal payloads for mobile
- **IoT BFF** (Go): Ultra-compact for embedded devices
- **Admin BFF** (Python): Complex analytics for dashboards

## Running

```bash
elide run server.ts
```

## Benefits

- Client-optimized APIs
- Reduced payload sizes
- Platform-specific features
- Independent evolution
