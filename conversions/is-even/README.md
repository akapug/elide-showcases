# is-even - Elide Polyglot Showcase

> **One is-even implementation for ALL languages**

## Features

- ✅ Works across TypeScript, Python, Ruby, and Java
- ✅ Zero dependencies
- ✅ High performance

## Quick Start

### TypeScript
```typescript
import module from './elide-is-even.ts';
```

### Python
```python
from elide import require
module = require('./elide-is-even.ts')
```

### Ruby
```ruby
module = Elide.require('./elide-is-even.ts')
```

### Java
```java
Value module = context.eval("js", "require('./elide-is-even.ts')");
```
