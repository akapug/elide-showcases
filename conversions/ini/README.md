# ini - Elide Polyglot Showcase

> **One ini implementation for ALL languages**

## Features

- ✅ Works across TypeScript, Python, Ruby, and Java
- ✅ Zero dependencies
- ✅ High performance

## Quick Start

### TypeScript
```typescript
import module from './elide-ini.ts';
```

### Python
```python
from elide import require
module = require('./elide-ini.ts')
```

### Ruby
```ruby
module = Elide.require('./elide-ini.ts')
```

### Java
```java
Value module = context.eval("js", "require('./elide-ini.ts')");
```
