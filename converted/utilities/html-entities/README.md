## Features

- ✅ Works across TypeScript, Python, Ruby, and Java
- ✅ Zero dependencies
- ✅ High performance

## Quick Start

### TypeScript
```typescript
import module from './elide-html-entities.ts';
```

### Python
```python
from elide import require
module = require('./elide-html-entities.ts')
```

### Ruby
```ruby
module = Elide.require('./elide-html-entities.ts')
```

### Java
```java
Value module = context.eval("js", "require('./elide-html-entities.ts')");
```
