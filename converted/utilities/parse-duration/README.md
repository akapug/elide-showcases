# parse-duration - Elide Polyglot Showcase

> **Parse duration strings for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Parse human-readable durations
- Support for multiple time units
- **~2M downloads/week on npm**

## Quick Start

```typescript
import parseDuration from './elide-parse-duration.ts';

parseDuration('5 seconds'); // 5000
parseDuration('2 hours'); // 7200000
parseDuration('1h 30m'); // 5400000
parseDuration('3 weeks'); // 1814400000
```
