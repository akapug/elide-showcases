# EditorConfig - Elide Polyglot Showcase

> **Consistent coding styles** - One config for all editors

EditorConfig helps maintain consistent coding styles across different editors.

## Features

- Cross-editor compatibility
- Simple configuration
- Language-agnostic
- **~500K downloads/week on npm**

## Quick Start

```typescript
import editorconfig from './elide-editorconfig.ts';

const formatted = editorconfig.format('const\tx = 10;');
console.log(formatted);
```
