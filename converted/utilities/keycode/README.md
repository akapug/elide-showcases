# keycode - Elide Polyglot Showcase

> **Key code utilities for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Key code to name conversion
- Name to key code conversion
- Event parsing
- Alias support
- **~500K+ downloads/week on npm**

## Quick Start

```typescript
import keycode from './elide-keycode.ts';

keycode(13); // 'enter'
keycode('escape'); // 27
keycode.isEventKey(event, 'enter');
```
