# Voca - Elide Polyglot Showcase

> **One modern string library for ALL languages** - TypeScript, Python, Ruby, and Java

Modern JavaScript string manipulation with comprehensive utilities.

## Features

- Case conversions (camel, kebab, snake, title)
- Truncate, trim, count
- Escape HTML, slugify
- And many more utilities
- ~500K+ downloads/week on npm

## Quick Start

```typescript
import { camelCase, slugify, countWords } from './elide-voca.ts';

camelCase("hello-world");         // "helloWorld"
slugify("Hello World!");          // "hello-world"
countWords("hello world test");   // 3
```
