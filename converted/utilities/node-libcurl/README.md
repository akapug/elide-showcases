# Node LibCURL

Node LibCURL for Elide (polyglot!)

Based on https://www.npmjs.com/package/node-libcurl (~50K+ downloads/week)

## Features

- cURL power
- High performance
- Protocol support
- Binary data
- Zero dependencies

## Quick Start

```typescript
import node-libcurl from './elide-node-libcurl.ts';

// Basic usage
const result = node-libcurl.main();
console.log(result);
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript

```
