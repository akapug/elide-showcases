# Unique Random

Generate unique random numbers for Elide (polyglot!)

Based on https://www.npmjs.com/package/unique-random (~100K+ downloads/week)

## Features

- Generate unique random numbers
- Easy to use API
- Type-safe
- Zero dependencies

## Quick Start

```typescript
import unique_random from './elide-unique-random.ts';

// Use the library
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript
import unique_random from './elide-unique-random.ts';
```

### Python (via Elide)
```python
from elide_unique_random import unique_random
```

### Ruby (via Elide)
```ruby
require 'elide/unique_random'
```

### Java (via Elide)
```java
import elide.unique_random.*;
```
