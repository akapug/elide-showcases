# regression-js - Elide Polyglot Showcase

> **Simple regression for ALL languages** - TypeScript, Python, R, and Julia

## Features

- Linear regression
- Simple API
- **~20K downloads/week on npm**

## Quick Start

```typescript
import { linearRegression } from './elide-regression-js.ts';

const data = [[1, 2], [2, 4], [3, 6]];
const model = linearRegression(data);
model.predict(4);
```
