# moment-business-days - Elide Polyglot Showcase

> **Business day operations for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Add/subtract business days
- Calculate business day differences
- **~1M downloads/week on npm**

## Quick Start

```typescript
import momentBusinessDays from './elide-moment-business-days.ts';

momentBusinessDays().isBusinessDay();
momentBusinessDays().businessDaysAdd(5);
momentBusinessDays().nextBusinessDay();
momentBusinessDays().businessDiff(new Date());
```
