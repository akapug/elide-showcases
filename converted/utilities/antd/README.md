- 50+ high-quality React components

- Full TypeScript support
- **~1.5M downloads/week on npm**

## Quick Start

```typescript
import { Button, Table, Message } from './elide-antd.ts';

// Render a button
Button.render({ type: 'primary', children: 'Click me' });

// Show notification
Message.success('Operation successful!');

// Render a table
Table.render({ columns, dataSource });
```
