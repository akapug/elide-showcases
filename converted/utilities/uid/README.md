# UID

Unique ID generator for Elide (polyglot!)

Based on https://www.npmjs.com/package/uid (~200K+ downloads/week)

## Features

- Unique ID generator
- Fast and efficient
- Type-safe
- Zero dependencies

## Quick Start

```typescript
import uid from './elide-uid.ts';
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript
import uid from './elide-uid.ts';
```

### Python (via Elide)
```python
from elide_uid import uid
```

### Ruby (via Elide)
```ruby
require 'elide/uid'
```

### Java (via Elide)
```java
import elide.uid.*;
```

## Benefits

- One uid library for ALL languages on Elide
