# kalman-filter - Elide Polyglot Showcase

> **Kalman filtering for ALL languages** - TypeScript, Python, R, and Julia

## Features

- 1D Kalman filter
- State estimation
- Noise reduction
- **~10K downloads/week on npm**

## Quick Start

```typescript
import KalmanFilter from './elide-kalman-filter.ts';

const kf = new KalmanFilter(0.01, 1);
kf.filter([1, 1.1, 0.9, 1.2]);
```
