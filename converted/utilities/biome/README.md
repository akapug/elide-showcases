# Biome - Elide Polyglot Showcase

> **Fast toolchain** - One tool for formatting and linting

One toolchain for your web project: formatter, linter, and more.

## Features

- Extremely fast (Rust-based)
- Format & lint in one tool
- TypeScript/JavaScript/JSON support
- **~100K downloads/week on npm**

## Quick Start

```typescript
import biome from './elide-biome.ts';

const result = biome.check('var x = 10');
console.log(result.formatted);
```
