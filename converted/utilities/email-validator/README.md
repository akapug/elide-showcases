# Email Validator - Elide Polyglot Showcase

> **One email validator for ALL languages** - TypeScript, Python, Ruby, and Java

Simple, fast email validation library.

## Features

- Fast email validation
- RFC 5322 compliant
- Zero dependencies
- **~5M downloads/week on npm**

## Quick Start

```typescript
import emailValidator from './elide-email-validator.ts';

console.log(emailValidator.validate("user@example.com")); // true
console.log(emailValidator.validate("not-an-email")); // false
```
