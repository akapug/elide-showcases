# Inferno - Elide Polyglot Showcase

> **One framework for ALL languages** - TypeScript, Python, Ruby, and Java

Blazing fast React-like library with fast rendering and small bundle size.

## Features

- Fast rendering
- Small bundle size
- Virtual DOM
- Component-based
- **~500K downloads/week on npm**

## Quick Start

```typescript
import Inferno from './elide-inferno.ts';

const vnode = Inferno.h('h1', null, 'Hello World');
const html = Inferno.render(vnode);
```
