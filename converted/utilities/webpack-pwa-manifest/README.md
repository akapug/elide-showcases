# webpack-pwa-manifest - Elide Polyglot Showcase

> **PWA manifests for ALL frameworks** - TypeScript, Python, Ruby, and Java

## Features

- Auto-generate manifest.json
- Icon generation
- Theme color support
- **~100K+ downloads/week on npm**

## Quick Start

```typescript
import { createPWAManifest } from './elide-webpack-pwa-manifest.ts';

const manifest = createPWAManifest({
  name: 'My PWA',
  theme_color: '#3f51b5'
});
```
