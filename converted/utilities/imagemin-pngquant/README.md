# imagemin-pngquant - Elide Polyglot Showcase

> **PNG Optimizer for Imagemin for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- PNG compression plugin
- Quality-based optimization
- Speed/quality trade-offs
- **~5M downloads/week on npm**

## Quick Start

```typescript
import imageminpngquant from './elide-imagemin-pngquant.ts';

const result = imageminPngquant({
  quality: [0.65, 0.80],
  speed: 1
});
console.log('PNG optimization ready');
```
