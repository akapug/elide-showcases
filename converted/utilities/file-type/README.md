# file-type - Elide Polyglot Showcase

> **Detect File Type for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Detect file type from buffer
- Support 100+ formats
- Magic number detection
- **~80M downloads/week on npm**

## Quick Start

```typescript
import filetype from './elide-file-type.ts';

const type = await fileType(buffer);
console.log('File type:', type?.ext, type?.mime);
```
