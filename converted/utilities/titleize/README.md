# Titleize - Elide Polyglot Showcase

> **One titleize library for ALL languages** - TypeScript, Python, Ruby, and Java

Convert strings to Title Case format.

## Features

- Capitalizes each word
- Handles camelCase, snake_case, kebab-case
- ~2M+ downloads/week on npm

## Quick Start

```typescript
import titleize from './elide-titleize.ts';
titleize("hello world");   // "Hello World"
titleize("helloWorld");    // "Hello World"
```
