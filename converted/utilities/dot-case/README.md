# Dot Case - Elide Polyglot Showcase

> **One dot.case library for ALL languages** - TypeScript, Python, Ruby, and Java

Convert strings to dot.case format for namespaces and object paths.

## Features

- Lowercase with dots
- Perfect for configuration keys
- ~2M+ downloads/week on npm

## Quick Start

```typescript
import dotCase from './elide-dot-case.ts';
dotCase("helloWorld");  // "hello.world"
```
