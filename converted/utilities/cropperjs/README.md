# cropperjs - Elide Polyglot Showcase

> **Advanced Image Cropper for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Touch-enabled crop interface
- Zoom, rotate, flip
- Responsive design
- **~3M downloads/week on npm**

## Quick Start

```typescript
import cropperjs from './elide-cropperjs.ts';

const cropper = new Cropper({
  aspectRatio: 16 / 9,
  viewMode: 1
});
console.log('Cropper ready');
```
