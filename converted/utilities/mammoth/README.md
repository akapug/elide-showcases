# mammoth - Elide Polyglot Showcase

> **DOCX to HTML conversion for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- DOCX to HTML conversion
- Style preservation
- Image extraction
- **Polyglot**: Use from TypeScript, Python, Ruby, Java
- Zero dependencies

## Quick Start

```typescript
import mammoth from './elide-mammoth.ts';

const result = await mammoth.convertToHtml({ path: 'document.docx' });
console.log(result.value);
```
