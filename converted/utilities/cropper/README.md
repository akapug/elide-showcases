# cropper - Elide Polyglot Showcase

> **Simple Image Cropper for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Lightweight image cropper
- Custom aspect ratios
- Mobile-friendly
- **~2M downloads/week on npm**

## Quick Start

```typescript
import cropper from './elide-cropper.ts';

const crop = new ImageCropper({
  width: 800,
  height: 600
});
console.log('Crop area: 800x600');
```
