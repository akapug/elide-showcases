# DOM Serializer - Elide Polyglot Showcase

> **One DOM serializer for ALL languages** - TypeScript, Python, Ruby, and Java

Serialize DOM trees back to HTML strings.

## Features

- DOM to HTML conversion
- Pretty printing
- Self-closing tags
- **~120M downloads/week on npm**

## Quick Start

```typescript
import DOMSerializer from './elide-dom-serializer.ts';

const html = DOMSerializer.render(domNodes);
console.log(html);
```
