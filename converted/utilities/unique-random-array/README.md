# Unique Random Array

Random array items without repeats for Elide (polyglot!)

Based on https://www.npmjs.com/package/unique-random-array (~200K+ downloads/week)

## Features

- Random array items without repeats
- Easy to use API
- Type-safe
- Zero dependencies

## Quick Start

```typescript
import unique_random_array from './elide-unique-random-array.ts';

// Use the library
```

## Polyglot Examples

```typescript
import unique_random_array from './elide-unique-random-array.ts';
```


### Ruby (via Elide)
