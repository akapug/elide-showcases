# styled-system - Elide Polyglot Showcase

> **Style props system - ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Style props
- Responsive values
- Theme-based
- **~300K downloads/week on npm**

## Quick Start

```typescript
import { space, color, compose } from './elide-styled-system.ts';

const styleProps = compose(space, color);
const styles = styleProps({ m: 2, color: 'blue' });
```
