# Particles.js

Particle effects for Elide (polyglot!)
Based on https://www.npmjs.com/package/particles.js (~50K+ downloads/week)

## Features
- Particle backgrounds
- Interactive modes
- Customizable
- Lightweight
- Zero dependencies

## Quick Start
```typescript
import particlesJS from './elide-particles.js.ts';
particlesJS('particles-js', { particles: { number: { value: 80 } } });
```
