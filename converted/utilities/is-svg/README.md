# is-svg - Elide Polyglot Showcase

> **Check if SVG for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Validate SVG files
- String/Buffer support
- Fast detection
- **~30M downloads/week on npm**

## Quick Start

```typescript
import issvg from './elide-is-svg.ts';

console.log('Is SVG string:', isSvg('<svg>...</svg>'));
console.log('Is SVG buffer:', isSvg(buffer));
```
