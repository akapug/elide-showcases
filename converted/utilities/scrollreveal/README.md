# ScrollReveal

Scroll animation library for Elide (polyglot!)

Based on https://www.npmjs.com/package/scrollreveal (~100K+ downloads/week)

## Features

- Scroll-triggered animations
- Customizable effects
- Sequential reveals
- Mobile support
- Easy configuration
- Zero dependencies

## Quick Start

```typescript
import ScrollReveal from './elide-scrollreveal.ts';

const sr = ScrollReveal();
sr.reveal('.element', {
  origin: 'left',
  distance: '50px',
  duration: 1000,
});
```
