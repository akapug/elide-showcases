# Fast UUID

Performance optimized UUID for Elide (polyglot!)

Based on https://www.npmjs.com/package/fast-uuid (~20K+ downloads/week)

## Features

- Performance optimized UUID
- Fast and efficient
- Type-safe
- Zero dependencies

## Quick Start

```typescript
import fast_uuid from './elide-fast-uuid.ts';
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript
import fast_uuid from './elide-fast-uuid.ts';
```

### Python (via Elide)
```python
from elide_fast_uuid import fast_uuid
```

### Ruby (via Elide)
```ruby
require 'elide/fast_uuid'
```

### Java (via Elide)
```java
import elide.fast_uuid.*;
```
