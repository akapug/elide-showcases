# UUID Random

Fast UUID v4 generation for Elide (polyglot!)

Based on https://www.npmjs.com/package/uuid-random (~100K+ downloads/week)

## Features

- Fast UUID v4 generation
- Fast and efficient
- Type-safe
- Zero dependencies

## Quick Start

```typescript
import uuid_random from './elide-uuid-random.ts';
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript
import uuid_random from './elide-uuid-random.ts';
```

### Python (via Elide)
```python
from elide_uuid_random import uuid_random
```

### Ruby (via Elide)
```ruby
require 'elide/uuid_random'
```

### Java (via Elide)
```java
import elide.uuid_random.*;
```
