# Decamelize - Elide Polyglot Showcase

> **One decamelize library for ALL languages** - TypeScript, Python, Ruby, and Java

Convert camelCase strings to other formats with configurable separators.

## Features

- Converts camelCase to snake_case
- Configurable separator
- Handles consecutive capitals
- ~30M+ downloads/week on npm

## Quick Start

```typescript
import decamelize from './elide-decamelize.ts';
decamelize("helloWorld");        // "hello_world"
decamelize("helloWorld", '-');   // "hello-world"
```
