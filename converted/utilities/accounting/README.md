# accounting - Elide Polyglot Showcase

> **One accounting library for ALL languages** - TypeScript, Python, Ruby, and Java

Simple and advanced number, money and currency formatting.

## Features

- Money formatting
- Currency formatting
- Number formatting
- Zero dependencies
- **~300K downloads/week on npm**

## Quick Start

```typescript
import { formatMoney } from './elide-accounting.ts';

console.log(formatMoney(1234.56)); // $1,234.56
```

## Documentation

Run the demo:

```bash
elide run elide-accounting.ts
```
