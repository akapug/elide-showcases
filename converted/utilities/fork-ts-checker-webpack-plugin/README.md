# fork-ts-checker-webpack-plugin - Elide Polyglot Showcase

> **Type checking for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Async type checking
- TypeScript and ESLint
- Fast incremental builds
- **~1M+ downloads/week on npm**

## Quick Start

```typescript
import ForkTsCheckerWebpackPlugin from './elide-fork-ts-checker-webpack-plugin.ts';

const plugin = new ForkTsCheckerWebpackPlugin({
  async: true,
  typescript: { enabled: true }
});
```
