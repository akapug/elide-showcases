# imagemin-svgo - Elide Polyglot Showcase

> **SVG Optimizer for Imagemin for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- SVG compression plugin
- Remove unnecessary data
- Optimize paths
- **~5M downloads/week on npm**

## Quick Start

```typescript
import imageminsvgo from './elide-imagemin-svgo.ts';

const result = imageminSvgo({
  plugins: [{ removeViewBox: false }]
});
console.log('SVG optimization ready');
```
