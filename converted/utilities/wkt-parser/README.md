# WKT Parser - Elide Polyglot Showcase

> **One WKT parser for ALL languages** - TypeScript, Python, Ruby, and Java

Parse and stringify Well-Known Text geometries.

## Features

- Parse WKT strings
- Convert to GeoJSON
- **~20K+ downloads/week on npm**

## Quick Start

```typescript
import { parse, stringify } from './elide-wkt-parser.ts';

const geom = parse("POINT(-74.006 40.7128)");
```
