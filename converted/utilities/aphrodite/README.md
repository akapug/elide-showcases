# aphrodite - Elide Polyglot Showcase

> **Inline styles - ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Inline styles
- Media queries
- Pseudo-selectors
- **~100K downloads/week on npm**

## Quick Start

```typescript
import { StyleSheet, css } from './elide-aphrodite.ts';

const styles = StyleSheet.create({
  button: { background: 'blue' }
});

const className = css(styles.button);
```
