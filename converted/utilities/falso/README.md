# Falso

Fake data generation for Elide (polyglot!)

Based on https://www.npmjs.com/package/falso (~20K+ downloads/week)

## Features

- Fake data generation
- Fast and efficient
- Type-safe
- Zero dependencies

## Quick Start

```typescript
import falso from './elide-falso.ts';
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript
import falso from './elide-falso.ts';
```

### Python (via Elide)
```python
from elide_falso import falso
```

### Ruby (via Elide)
```ruby
require 'elide/falso'
```

### Java (via Elide)
```java
import elide.falso.*;
```
