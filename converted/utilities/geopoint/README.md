# GeoPoint - Elide Polyglot Showcase

> **One GeoPoint class for ALL languages** - TypeScript, Python, Ruby, and Java

Class for working with geographic points and calculations.

## Quick Start

```typescript
import GeoPoint from './elide-geopoint.ts';

const p1 = new GeoPoint(40.7128, -74.0060);
const p2 = new GeoPoint(51.5074, -0.1278);
console.log(p1.distanceTo(p2)); // km
```
