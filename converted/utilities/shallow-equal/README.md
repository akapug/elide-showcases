# Shallow Equal - Shallow Equality Comparison

Fast shallow equality for objects and arrays on Elide.

## Quick Start

```typescript
import shallowEqual from './elide-shallow-equal.ts';

console.log(shallowEqual({ a: 1 }, { a: 1 })); // true
```
