# Humanize URL - Elide Polyglot Showcase

> **One humanize url for ALL languages** - TypeScript, Python, Ruby, and Java

Convert URLs to human-friendly format by removing protocol and trailing slashes

## Features

- Pure TypeScript implementation
- Zero dependencies
- Fully typed interfaces
- Production-ready
- Polyglot: Use from TypeScript, Python, Ruby, and Java
