## Features

- 100+ customizable components
- TypeScript based
- Dark theme support
- **~200K downloads/week on npm**

## Quick Start

```typescript
import { Button, TextInput, notifications } from './elide-mantine.ts';

Button.render({ variant: 'filled', children: 'Click' });
TextInput.render({ label: 'Email', required: true });
notifications.show({ message: 'Saved!', color: 'green' });
```
