## Quick Start

### TypeScript
```typescript
import module from './elide-group-by.ts';
```

### Python
```python
from elide import require
module = require('./elide-group-by.ts')
```

### Ruby
```ruby
module = Elide.require('./elide-group-by.ts')
```

### Java
```java
Value module = context.eval("js", "require('./elide-group-by.ts')");
```
