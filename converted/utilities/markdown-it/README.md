# markdown-it - Elide Polyglot Showcase

> **Markdown parsing for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- CommonMark compliant
- Extensible plugins
- Syntax highlighting
- **Polyglot**: Use from TypeScript, Python, Ruby, Java
- Zero dependencies

## Quick Start

```typescript
import markdownit from './elide-markdown-it.ts';

const md = markdownit();
const html = md.render('# Hello\n\nThis is **bold**.');
console.log(html);
```
