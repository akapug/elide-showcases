# React Router Native - Elide Polyglot Showcase

> **One react router native for ALL languages** - TypeScript, Python, Ruby, and Java

React Native bindings for React Router with native navigation

## Features

- Pure TypeScript implementation
- Zero dependencies
- Fully typed interfaces
- Production-ready
- Polyglot: Use from TypeScript, Python, Ruby, and Java

## Quick Start

### TypeScript
```typescript
import lib from './elide-react-router-native.ts';

// Your code here
```

### Python
```python
from elide import require
lib = require('./elide-react-router-native.ts')

# Your code here
```

### Ruby
```ruby
lib = Elide.require('./elide-react-router-native.ts')

# Your code here
```

### Java
```java
Value lib = context.eval("js", "require('./elide-react-router-native.ts')");

// Your code here
```

## Why Polyglot?

Write once in TypeScript, use everywhere:
- **Consistency**: Same behavior across all languages
- **Maintainability**: Single implementation to maintain
- **Type Safety**: Full TypeScript types available
- **Performance**: Native Elide performance
