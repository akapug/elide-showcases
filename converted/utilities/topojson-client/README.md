# TopoJSON Client - Elide Polyglot Showcase

> **One TopoJSON client for ALL languages** - TypeScript, Python, Ruby, and Java

Convert TopoJSON to GeoJSON.

## Quick Start

```typescript
import { feature } from './elide-topojson-client.ts';

const geojson = feature(topology, topology.objects.countries);
```
