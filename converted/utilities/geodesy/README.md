# Geodesy - Elide Polyglot Showcase

> **One geodesy library for ALL languages** - TypeScript, Python, Ruby, and Java

Precise geodetic calculations for navigation and surveying.

## Quick Start

```typescript
import LatLon from './elide-geodesy.ts';

const p1 = new LatLon(40.7128, -74.0060);
const p2 = new LatLon(51.5074, -0.1278);
console.log(p1.distanceTo(p2));
```
