# holidays - Elide Polyglot Showcase

> **Holiday calculator for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Calculate holiday dates
- US holidays support
- **~500K downloads/week on npm**

## Quick Start

```typescript
import holidays from './elide-holidays.ts';

holidays.getHolidays(2025); // Get all holidays for 2025
holidays.isHoliday(new Date(2025, 11, 25)); // Check if date is a holiday
```
