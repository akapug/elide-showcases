# Stylelint Config Prettier - Elide Polyglot Showcase

> **Stylelint + Prettier** - Turn off conflicting rules

Turns off all Stylelint rules that conflict with Prettier.

## Features

- Disable conflicting rules
- Use with Prettier
- **~300K downloads/week on npm**

## Quick Start

```typescript
import prettierConfig from './elide-stylelint-config-prettier.ts';

const config = prettierConfig.getConfig();
```
