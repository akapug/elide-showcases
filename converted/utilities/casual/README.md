# Casual

Fake data generator for Elide (polyglot!)

Based on https://www.npmjs.com/package/casual (~100K+ downloads/week)

## Features

- Fake data generator
- Fast and efficient
- Type-safe
- Zero dependencies

## Quick Start

```typescript
import casual from './elide-casual.ts';
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript
import casual from './elide-casual.ts';
```

### Python (via Elide)
```python
from elide_casual import casual
```

### Ruby (via Elide)
```ruby
require 'elide/casual'
```

### Java (via Elide)
```java
import elide.casual.*;
```
