# calendar - Elide Polyglot Showcase

> **Calendar utilities for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Generate calendar grids
- Month/year navigation
- **~1M downloads/week on npm**

## Quick Start

```typescript
import Calendar from './elide-calendar.ts';

const cal = new Calendar(2025, 10); // November 2025
cal.monthDays(); // Get all days in month as weeks
cal.toString(); // "November 2025"
```
