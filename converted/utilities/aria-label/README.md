# aria-label - Elide Polyglot Showcase

> **ARIA labels for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Label management
- Labelledby support
- Auto ID generation
- **~30K+ downloads/week on npm**

## Quick Start

```typescript
import ariaLabel from './elide-aria-label.ts';

ariaLabel.setLabel(button, 'Close');
ariaLabel.setLabelledBy(input, 'label-id');
```
