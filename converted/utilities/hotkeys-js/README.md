# hotkeys-js - Elide Polyglot Showcase

> **Hotkeys management for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Simple hotkey binding
- Scope management
- Filter callbacks
- Modifier combinations
- **~200K+ downloads/week on npm**

## Quick Start

```typescript
import hotkeys from './elide-hotkeys-js.ts';

hotkeys.bind('ctrl+s', () => console.log('Save!'));
hotkeys.setScope('editor');
```
