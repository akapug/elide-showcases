# frequency-tools - Elide Polyglot Showcase

> **Frequency utilities for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Frequency to note conversion
- Note to frequency conversion
- Cent calculations
- **~3K+ downloads/week on npm**

## Quick Start

```typescript
import { frequencyToNote, noteToFrequency } from './elide-frequency-tools.ts';

const freq = noteToFrequency('A', 4); // 440Hz
const { note, octave } = frequencyToNote(440);
```
