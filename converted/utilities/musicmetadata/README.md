# musicmetadata - Elide Polyglot Showcase

> **Metadata extraction for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Parse audio metadata
- ID3 tag support
- Album art extraction
- **~30K+ downloads/week on npm**

## Quick Start

```typescript
import parse from './elide-musicmetadata.ts';

parse(stream, (err, metadata) => {
  console.log(metadata.title);
});
```
