# Blade - Elide Polyglot Showcase

> **One Blade engine for ALL languages** - TypeScript, Python, Ruby, and Java

Laravel's powerful templating engine ported to JavaScript.

## Features

- Blade syntax support
- Template inheritance
- Control structures
- **~300K downloads/week on npm**

## Quick Start

```typescript
import Blade from './elide-blade.ts';

const blade = new Blade();
const html = blade.render('Hello {{ name }}!', { name: 'World' });
```
