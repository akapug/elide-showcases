# Prettier Plugin Sort Imports - Elide Polyglot Showcase

> **Sort imports** - Sort import declarations by custom order

A prettier plugin to sort import declarations by provided RegEx order.

## Features

- Sort imports by custom order
- Group imports
- Remove duplicates
- **~50K downloads/week on npm**

## Quick Start

```typescript
import sortPlugin from './elide-prettier-plugin-sort-imports.ts';

const sorted = sortPlugin.sort(code);
```
