# tinytime - Elide Polyglot Showcase

> **Tiny date formatter for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Ultra-lightweight date formatting
- Template-based formatting
- **~500K downloads/week on npm**

## Quick Start

```typescript
import tinytime from './elide-tinytime.ts';

const template = tinytime('{YYYY}-{MM}-{DD} {h}:{mm}{a}');
template.render(new Date()); // "2025-11-18 3:45pm"
```
