# showdown - Elide Polyglot Showcase

> **Markdown/HTML conversion for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Markdown to HTML
- HTML to Markdown
- GitHub Flavored Markdown
- **Polyglot**: Use from TypeScript, Python, Ruby, Java
- Zero dependencies

## Quick Start

```typescript
import { Converter } from './elide-showdown.ts';

const converter = new Converter();
const html = converter.makeHtml('# Hello');
const md = converter.makeMarkdown('<h1>Hello</h1>');
```
