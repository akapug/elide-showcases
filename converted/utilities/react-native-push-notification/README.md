# React Native Push Notification - Elide Polyglot Showcase

> **One push notification library for ALL languages** - TypeScript, Python, Ruby, and Java

Local and remote push notifications for React Native.

## Features

- Local notifications
- Remote notifications
- Scheduled notifications
- Badges
- Sounds
- **~300K downloads/week on npm**

## Quick Start

```typescript
import PushNotification from './elide-react-native-push-notification.ts';

PushNotification.localNotification({
  title: 'Hello',
  message: 'This is a notification!',
});
```
