# headlessui - Elide Polyglot Showcase

> **Unstyled accessible components for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Completely unstyled
- Fully accessible
- **~300K downloads/week on npm**

## Quick Start

```typescript
import { Dialog, Menu } from './elide-headlessui.ts';

Dialog.render({ open: true, children: 'Content' });
Menu.Button({ children: 'Menu' });
```
