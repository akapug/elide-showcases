# pixelmatch - Elide Polyglot Showcase

> **Pixel-level Image Comparison for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Compare images pixel by pixel
- Diff visualization
- Anti-aliasing detection
- **~15M downloads/week on npm**

## Quick Start

```typescript
import pixelmatch from './elide-pixelmatch.ts';

const diff = pixelmatch(img1, img2, null, 800, 600);
console.log('Different pixels:', diff);
```
