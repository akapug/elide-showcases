# tsParticles

TypeScript particles for Elide (polyglot!)
Based on https://www.npmjs.com/package/tsparticles (~100K+ downloads/week)

## Features
- Particle effects
- Interactive modes
- Customizable
- Multiple shapes
- Zero dependencies

## Quick Start
```typescript
import { loadFull } from './elide-tsparticles.ts';
await loadFull(container);
```
