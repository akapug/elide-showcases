## Quick Start

```typescript
import canvas from './elide-canvas.ts';

const canvas = createCanvas(800, 600);
const ctx = canvas.getContext('2d');
ctx.fillStyle = 'blue';
ctx.fillRect(10, 10, 100, 100);
```
