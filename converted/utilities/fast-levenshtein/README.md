# Fast Levenshtein - Elide Polyglot Showcase

> **One Levenshtein distance library for ALL languages** - TypeScript, Python, Ruby, and Java

Calculate edit distance between strings efficiently.

## Features

- Optimized Levenshtein algorithm
- Low memory usage
- Perfect for spell checking, fuzzy matching
- ~40M+ downloads/week on npm

## Quick Start

```typescript
import levenshtein from './elide-fast-levenshtein.ts';
levenshtein("kitten", "sitting");  // 3
levenshtein("hello", "hallo");     // 1
```
