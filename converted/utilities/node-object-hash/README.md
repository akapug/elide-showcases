# Node Object Hash

Fast consistent object hashing for Elide (polyglot!)

Based on https://www.npmjs.com/package/node-object-hash (~100K+ downloads/week)

## Features

- Fast consistent object hashing
- Fast hashing algorithm
- Deterministic output
- Zero dependencies

## Quick Start

```typescript
import node_object_hash from './elide-node-object-hash.ts';
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript
import node_object_hash from './elide-node-object-hash.ts';
```

### Python (via Elide)
```python
from elide_node_object_hash import node_object_hash
```

### Ruby (via Elide)
```ruby
require 'elide/node_object_hash'
```

### Java (via Elide)
```java
import elide.node_object_hash.*;
```
