# Split Lines - Elide Polyglot Showcase

> **One line splitting library for ALL languages** - TypeScript, Python, Ruby, and Java

Split text into lines while handling all newline types.

## Features

- Handles \n, \r\n, and \r
- Option to preserve newline characters
- ~5M+ downloads/week on npm

## Quick Start

```typescript
import splitLines from './elide-split-lines.ts';

splitLines("hello\nworld\r\ntest");  // ["hello", "world", "test"]
```
