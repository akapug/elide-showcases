# enzyme-adapter-react-16 - Elide Polyglot Showcase

> **One React adapter for ALL languages** - TypeScript, Python, Ruby, and Java

Enzyme adapter for React 16.

## Features

- React 16 compatibility
- Hooks support
- Fragment support
- Portal support
- Zero dependencies
- **~800K downloads/week on npm**

## Quick Start

```typescript
import Adapter from './elide-enzyme-adapter-react-16.ts';
import { configure } from 'enzyme';

configure({ adapter: new Adapter() });
```
