# human-interval - Elide Polyglot Showcase

> **Human-readable intervals for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Parse human-readable time intervals
- Natural language support
- **~1M downloads/week on npm**

## Quick Start

```typescript
import humanInterval from './elide-human-interval.ts';

humanInterval('3 seconds'); // 3000
humanInterval('5 minutes'); // 300000
humanInterval('2 hours'); // 7200000
humanInterval('day'); // 86400000
```
