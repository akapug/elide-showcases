# SlugID

URL-safe base64 UUIDs for Elide (polyglot!)

Based on https://www.npmjs.com/package/slugid (~50K+ downloads/week)

## Features

- URL-safe base64 UUIDs
- Fast and efficient
- Type-safe
- Zero dependencies

## Quick Start

```typescript
import slugid from './elide-slugid.ts';
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript
import slugid from './elide-slugid.ts';
```

### Python (via Elide)
```python
from elide_slugid import slugid
```

### Ruby (via Elide)
```ruby
require 'elide/slugid'
```

### Java (via Elide)
```java
import elide.slugid.*;
```

## Benefits

- One slugid library for ALL languages on Elide
