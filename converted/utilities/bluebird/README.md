# Bluebird for Elide

Full-featured promise library with extra utility methods.

**Downloads**: ~7M/week on npm

## Quick Start

```typescript
import Promise from './bluebird.ts';

await Promise.delay(1000);
const results = await Promise.map([1, 2, 3], async x => x * 2);
```
