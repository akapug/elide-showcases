# Node Geocoder - Elide Polyglot Showcase

> **One geocoding service for ALL languages** - TypeScript, Python, Ruby, and Java

Interface for multiple geocoding providers.

## Quick Start

```typescript
import NodeGeocoder from './elide-node-geocoder.ts';

const geocoder = new NodeGeocoder();
const results = await geocoder.geocode('New York');
```
