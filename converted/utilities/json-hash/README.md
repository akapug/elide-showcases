# JSON Hash

Hash JSON objects consistently for Elide (polyglot!)

Based on https://www.npmjs.com/package/json-hash (~50K+ downloads/week)

## Features

- Hash JSON objects consistently
- Fast hashing algorithm
- Deterministic output
- Zero dependencies

## Quick Start

```typescript
import json_hash from './elide-json-hash.ts';
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript
import json_hash from './elide-json-hash.ts';
```

### Python (via Elide)
```python
from elide_json_hash import json_hash
```

### Ruby (via Elide)
```ruby
require 'elide/json_hash'
```

### Java (via Elide)
```java
import elide.json_hash.*;
```
