# color2k - Elide Polyglot Showcase

> **Color manipulation - ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Parse colors
- Lighten/darken
- Mix colors
- **~300K downloads/week on npm**

## Quick Start

```typescript
import { lighten, darken, rgba } from './elide-color2k.ts';

const light = lighten('#007bff', 0.2);
const dark = darken('#007bff', 0.2);
```
