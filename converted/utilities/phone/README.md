# Phone - Elide Polyglot Showcase

> **One phone validator for ALL languages** - TypeScript, Python, Ruby, and Java

International phone number validation and formatting.

## Features

- International phone validation
- Number formatting
- Zero dependencies
- **~500K downloads/week on npm**

## Quick Start

```typescript
import phone from './elide-phone.ts';

console.log(phone.validate("(555) 123-4567")); // true
console.log(phone.format("5551234567")); // (555) 123-4567
```
