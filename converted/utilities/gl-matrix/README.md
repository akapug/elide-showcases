# gl-matrix - WebGL Math Library

High-performance matrix and vector operations for WebGL.

**POLYGLOT SHOWCASE**: Fast 3D math in ANY language on Elide!

## Quick Start

```typescript
import { vec3, mat4 } from './elide-gl-matrix.ts';

const v = vec3.fromValues(1, 2, 3);
const matrix = mat4.create();
mat4.translate(matrix, matrix, v);
```
