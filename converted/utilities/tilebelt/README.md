# Tilebelt - Elide Polyglot Showcase

> **One tile utility for ALL languages** - TypeScript, Python, Ruby, and Java

Tile coordinate math and utilities for map tiles.

## Quick Start

```typescript
import { tileToBBOX, getParent, getChildren } from './elide-tilebelt.ts';

const bbox = tileToBBOX([0, 0, 0]);
const children = getChildren([0, 0, 0]);
```
