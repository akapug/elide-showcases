# NUID

Highly performant unique identifier for Elide (polyglot!)

Based on https://www.npmjs.com/package/nuid (~100K+ downloads/week)

## Features

- Highly performant unique identifier
- Fast and efficient
- Type-safe
- Zero dependencies

## Quick Start

```typescript
import nuid from './elide-nuid.ts';
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript
import nuid from './elide-nuid.ts';
```

### Python (via Elide)
```python
from elide_nuid import nuid
```

### Ruby (via Elide)
```ruby
require 'elide/nuid'
```

### Java (via Elide)
```java
import elide.nuid.*;
```

## Benefits

- One nuid library for ALL languages on Elide
