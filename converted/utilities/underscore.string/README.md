# Underscore.string - String Utilities

String manipulation extensions.

Based on [underscore.string](https://www.npmjs.com/package/underscore.string) (~200K+ downloads/week)

```typescript
import _s from './elide-underscore.string.ts';

_s.capitalize("hello"); // "Hello"
_s.camelize("hello-world"); // "helloWorld"
_s.slugify("Hello World!"); // "hello-world"
```

Run: `elide run elide-underscore.string.ts`
