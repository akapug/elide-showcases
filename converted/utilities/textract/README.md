# textract - Elide Polyglot Showcase

> **Text extraction for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Extract from PDF, DOCX, PPTX, images
- OCR support
- Multiple formats
- **Polyglot**: Use from TypeScript, Python, Ruby, Java
- Zero dependencies

## Quick Start

```typescript
import textract from './elide-textract.ts';

textract.fromFileWithPath('document.pdf', (err, text) => {
  console.log(text);
});
```
