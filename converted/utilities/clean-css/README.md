# CleanCSS - Elide Polyglot Showcase

> **One CSS optimizer for ALL languages** - TypeScript, Python, Ruby, and Java

Fast and efficient CSS minifier and optimizer.

## Features

- CSS minification
- Property optimization
- **~30M downloads/week on npm**

## Quick Start

```typescript
import CleanCSS from './elide-clean-css.ts';

const cleanCSS = new CleanCSS();
const result = cleanCSS.minify('body { margin: 0; }');
console.log(result.styles);
```
