# clean-webpack-plugin - Elide Polyglot Showcase

> **Build cleaning for ALL build systems** - TypeScript, Python, Ruby, and Java

## Features

- Clean output directory
- Safe cleaning
- Dry run mode
- **~1M+ downloads/week on npm**

## Quick Start

```typescript
import CleanWebpackPlugin from './elide-clean-webpack-plugin.ts';

const plugin = new CleanWebpackPlugin({
  verbose: true
});

plugin.cleanBeforeBuild('dist');
```
