# CUID2

Next generation CUIDs for Elide (polyglot!)

Based on https://www.npmjs.com/package/cuid2 (~50K+ downloads/week)

## Features

- Next generation CUIDs
- Unique and sortable
- Fast generation
- Zero dependencies

## Quick Start

```typescript
import cuid2 from './elide-cuid2.ts';
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript
import cuid2 from './elide-cuid2.ts';
```

### Python (via Elide)
```python
from elide_cuid2 import cuid2
```

### Ruby (via Elide)
```ruby
require 'elide/cuid2'
```

### Java (via Elide)
```java
import elide.cuid2.*;
```
