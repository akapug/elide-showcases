# combokeys - Elide Polyglot Showcase

> **Keyboard shortcuts for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Key combination handling
- Sequence support
- Element-specific binding
- **~50K+ downloads/week on npm**

## Quick Start

```typescript
import Combokeys from './elide-combokeys.ts';

const combokeys = new Combokeys();
combokeys.bind('ctrl+s', () => save());
```
