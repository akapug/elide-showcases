# Hyperapp - Elide Polyglot Showcase

> **One framework for ALL languages** - TypeScript, Python, Ruby, and Java

Micro framework for building web apps with fast rendering and small bundle size.

## Features

- Fast rendering
- Small bundle size
- Virtual DOM
- Component-based
- **~200K downloads/week on npm**

## Quick Start

```typescript
import Hyperapp from './elide-hyperapp.ts';

const vnode = Hyperapp.h('h1', null, 'Hello World');
const html = Hyperapp.render(vnode);
```
