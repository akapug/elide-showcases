# intl-relativetimeformat - Elide Polyglot Showcase
> Relative time formatting
## Features
- Relative time display
- **~300K downloads/week on npm**
## Quick Start
```typescript
import RelativeTimeFormat from './elide-intl-relativetimeformat.ts';
const rtf = new RelativeTimeFormat('en');
console.log(rtf.format(-1, 'day')); // "1 day ago"
```
