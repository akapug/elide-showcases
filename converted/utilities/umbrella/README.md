# Umbrella - Elide Polyglot Showcase

> **One minimal jQuery alternative for ALL languages** - TypeScript, Python, Ruby, and Java

Tiny library for DOM manipulation and events.

## Features

- Ultra-small size (~3KB)
- jQuery-like syntax
- DOM manipulation
- Event handling
- **~100K downloads/week on npm**

## Quick Start

```typescript
import u from './elide-umbrella.ts';

u('div').addClass('active');
u('.item').html('<span>Hello</span>');
u('button').on('click', () => console.log('Clicked!'));
u('.box').trigger('change');
```
