# Slick Carousel

Responsive carousel for Elide (polyglot!)
Based on https://www.npmjs.com/package/slick-carousel (~500K+ downloads/week)

## Features
- Responsive breakpoints
- Infinite loop
- Autoplay
- Dots/arrows navigation
- Multiple slides
- Zero dependencies

## Quick Start
```typescript
import Slick from './elide-slick-carousel.ts';
const slick = new Slick(element, { slidesToShow: 3, autoplay: true });
```
