# colord - Elide Polyglot Showcase

> **Fast color manipulation - ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Fast and lightweight
- Color parsing
- Manipulation methods
- **~500K downloads/week on npm**

## Quick Start

```typescript
import { colord } from './elide-colord.ts';

const color = colord('#ff0000');
console.log(color.toHex());
console.log(color.lighten(0.1).toRgbString());
```
