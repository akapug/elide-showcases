# descriptive-statistics - Elide Polyglot Showcase

> **Descriptive statistics for ALL languages** - TypeScript, Python, R, and Julia

## Features

- Central tendency (mean, median)
- Dispersion (variance, std dev)
- Shape (skewness, kurtosis)
- **~5K downloads/week on npm**

## Quick Start

```typescript
import { mean, skewness, kurtosis } from './elide-descriptive-statistics.ts';

const data = [1, 2, 3, 4, 5];
mean(data);
skewness(data);
```
