# color-convert - Elide Polyglot Showcase

> **Color space conversion for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Convert between RGB, HEX, HSL, HSV
- **~150M downloads/week on npm**

## Quick Start

```typescript
import convert from './elide-color-convert.ts';

convert.rgb.hex(255, 87, 51); // 'ff5733'
convert.hex.rgb('FF5733'); // [255, 87, 51]
convert.rgb.hsl(255, 87, 51); // [h, s, l]
```
