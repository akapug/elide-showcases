# lunar - Elide Polyglot Showcase

> **Lunar calendar for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Solar to lunar date conversion
- Lunar to solar date conversion
- **~200K downloads/week on npm**

## Quick Start

```typescript
import lunar from './elide-lunar.ts';

lunar.getLunarDate(new Date()); // Convert to lunar date
lunar.getSolarDate({ year: 2025, month: 1, day: 1, isLeapMonth: false });
lunar.getLunarNewYear(2025); // Get Lunar New Year date
```
