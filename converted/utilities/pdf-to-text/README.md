# pdf-to-text - Elide Polyglot Showcase

> **PDF text extraction for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Extract text from PDFs
- Page-by-page extraction
- Simple API
- **Polyglot**: Use from TypeScript, Python, Ruby, Java
- Zero dependencies

## Quick Start

```typescript
import pdfToText from './elide-pdf-to-text.ts';

pdfToText('document.pdf', (err, text) => {
  console.log(text);
});
```
