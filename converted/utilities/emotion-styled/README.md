- styled.div`` syntax
- Prop-based styling
- Theme support

## Quick Start

```typescript
import styled from './elide-emotion-styled.ts';

const Button = styled.button`
  background: palevioletred;
  color: white;
  padding: 0.5em 1em;
`;
```
