# Path-to-RegExp - Elide Polyglot Showcase

> **One path-to-regexp for ALL languages** - TypeScript, Python, Ruby, and Java

Turn path strings into regular expressions with parameter extraction

## Features

- Pure TypeScript implementation
- Zero dependencies
- Fully typed interfaces
- Production-ready
- Polyglot: Use from TypeScript, Python, Ruby, and Java

## Quick Start

### TypeScript
```typescript
import lib from './elide-path-to-regexp.ts';

// Your code here
```

### Python
```python
from elide import require
lib = require('./elide-path-to-regexp.ts')

# Your code here
```

### Ruby
```ruby
lib = Elide.require('./elide-path-to-regexp.ts')

# Your code here
```

### Java
```java
Value lib = context.eval("js", "require('./elide-path-to-regexp.ts')");

// Your code here
```
