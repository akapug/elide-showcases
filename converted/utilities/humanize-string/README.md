# Humanize String - Elide Polyglot Showcase

> **One humanize library for ALL languages** - TypeScript, Python, Ruby, and Java

Convert code-style strings to human-readable format.

## Features

- Converts camelCase to readable format
- Handles snake_case and kebab-case
- Capitalizes appropriately
- ~3M+ downloads/week on npm

## Quick Start

```typescript
import humanizeString from './elide-humanize-string.ts';
humanizeString("helloWorld");  // "Hello world"
humanizeString("foo_bar");     // "Foo bar"
```
