# image-minimizer-webpack-plugin - Elide Polyglot Showcase

> **Image optimization for ALL web frameworks** - TypeScript, Python, Ruby, and Java

## Features

- PNG, JPEG, GIF, SVG optimization
- Configurable quality settings
- Parallel processing
- **~100K+ downloads/week on npm**

## Quick Start

```typescript
import ImageMinimizerWebpackPlugin from './elide-image-minimizer-webpack-plugin.ts';

const plugin = new ImageMinimizerWebpackPlugin({
  minimizer: {
    options: { quality: 80 }
  }
});
```
