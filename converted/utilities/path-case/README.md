# Path Case - Elide Polyglot Showcase

> **One path/case library for ALL languages** - TypeScript, Python, Ruby, and Java

Convert strings to path/case format for file system paths.

## Features

- Lowercase with slashes
- Perfect for file paths
- ~2M+ downloads/week on npm

## Quick Start

```typescript
import pathCase from './elide-path-case.ts';
pathCase("helloWorld");  // "hello/world"
```
