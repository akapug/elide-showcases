# Trim Newlines - Elide Polyglot Showcase

> **One newline trimming library for ALL languages** - TypeScript, Python, Ruby, and Java

Trim newline characters from the start and/or end of strings.

## Features

- Trim from start, end, or both
- Handles \n, \r\n, and \r
- ~30M+ downloads/week on npm

## Quick Start

```typescript
import trimNewlines from './elide-trim-newlines.ts';

trimNewlines("\n\nhello\n\n");  // "hello"
```
