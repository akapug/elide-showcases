# keyboard-key - Elide Polyglot Showcase

> **Keyboard event utilities for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Key code normalization
- Cross-browser compatibility
- Key name to code mapping
- Modifier key detection
- **~100K+ downloads/week on npm**

## Quick Start

```typescript
import keyboardKey from './elide-keyboard-key.ts';

const code = keyboardKey.toCode('Enter');
const key = keyboardKey.isKey(event, 'Escape');
```
