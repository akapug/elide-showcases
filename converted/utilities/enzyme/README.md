# enzyme - Elide Polyglot Showcase

> **One React testing utility for ALL languages** - TypeScript, Python, Ruby, and Java

JavaScript Testing utilities for React.

## Features

- Shallow rendering
- Full DOM rendering
- Static rendering
- Component traversal
- Manipulation APIs
- Zero dependencies
- **~1M downloads/week on npm**

## Quick Start

```typescript
import { shallow } from './elide-enzyme.ts';

const wrapper = shallow(<MyComponent />);
expect(wrapper.find('button').text()).toBe('Click');
```
