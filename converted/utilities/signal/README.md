# signal - Elide Polyglot Showcase

> **Signal processing for ALL languages** - TypeScript, Python, R, and Julia

## Features

- Moving average
- Peak detection
- Normalization
- **~20K downloads/week on npm**

## Quick Start

```typescript
import { movingAverage, findPeaks } from './elide-signal.ts';

const signal = [1, 3, 2, 5, 4, 3, 6, 2];
movingAverage(signal, 3);
findPeaks(signal, 3);
```
