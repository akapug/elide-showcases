# jsx-a11y - Elide Polyglot Showcase

> **JSX accessibility utilities for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- JSX accessibility rules
- ARIA attribute validation
- Interactive element checks
- Alt text validation
- **~500K+ downloads/week on npm**

## Quick Start

```typescript
import jsxA11y from './elide-jsx-a11y.ts';

const element = { type: 'button', props: {} };
const result = jsxA11y.validate(element);
```
