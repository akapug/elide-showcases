# aria-hidden - Elide Polyglot Showcase

> **Hide content from screen readers for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Hide elements from assistive technology
- Preserve focus management
- Automatic cleanup
- Modal dialog support
- **~500K+ downloads/week on npm**

## Quick Start

```typescript
import ariaHidden from './elide-aria-hidden.ts';

const undo = ariaHidden.hideOthers('modal-content');
// Later: undo();
```
