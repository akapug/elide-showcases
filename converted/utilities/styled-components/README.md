# styled-components - Elide Polyglot Showcase

> **CSS-in-JS for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Tagged template literals for component styles
- Dynamic styling with props
- Theming support
- **~3M downloads/week on npm**

## Quick Start

```typescript
import styled from './elide-styled-components.ts';

const Button = styled.button`
  background: palevioletred;
  color: white;
  padding: 0.25em 1em;
  border-radius: 3px;
`;

const button = Button({ children: 'Click Me' });
```
