# MP4 Parser - Elide Polyglot Showcase

> **One MP4 parser for ALL languages** - TypeScript, Python, Ruby, and Java

Parse MP4/MOV container format and extract atoms.

## Features

- MP4/MOV parsing
- Atom/box extraction
- Metadata reading
- **~10K downloads/week on npm**

## Quick Start

```typescript
import MP4Parser from './elide-mp4-parser.ts';

const parser = new MP4Parser();
const atoms = parser.parse(buffer);
```
