# easter-date - Elide Polyglot Showcase

> **Easter date calculator for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Calculate Easter Sunday for any year
- Related dates (Good Friday, Pentecost)
- **~100K downloads/week on npm**

## Quick Start

```typescript
import easterCalculator from './elide-easter-date.ts';

easterCalculator.easterDate(2025); // Easter Sunday
easterCalculator.goodFriday(2025); // Good Friday
easterCalculator.pentecost(2025); // Pentecost Sunday
```
