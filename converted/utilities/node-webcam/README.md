# Node Webcam - Elide Polyglot Showcase

> **One webcam library for ALL languages** - TypeScript, Python, Ruby, and Java

Capture images and video from webcams.

## Features

- Webcam capture
- Image snapshots
- Video recording
- Multiple cameras
- **~20K downloads/week on npm**

## Quick Start

```typescript
import Webcam from './elide-node-webcam.ts';

const webcam = new Webcam({ width: 1920, height: 1080 });
webcam.capture('photo.jpg');
```
