# Param Case - Elide Polyglot Showcase

> **One param-case library for ALL languages** - TypeScript, Python, Ruby, and Java

Convert strings to param-case format (alias for kebab-case).

## Features

- Lowercase with hyphens
- Perfect for URL parameters
- ~15M+ downloads/week on npm

## Quick Start

```typescript
import paramCase from './elide-param-case.ts';
paramCase("helloWorld");  // "hello-world"
```
