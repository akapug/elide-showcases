# Faker

Generate massive amounts of fake data for Elide (polyglot!)

Based on https://www.npmjs.com/package/faker (~3M+ downloads/week)

## Features

- Generate massive amounts of fake data
- Fast and efficient
- Type-safe
- Zero dependencies

## Quick Start

```typescript
import faker from './elide-faker.ts';
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript
import faker from './elide-faker.ts';
```

### Python (via Elide)
```python
from elide_faker import faker
```

### Ruby (via Elide)
```ruby
require 'elide/faker'
```

### Java (via Elide)
```java
import elide.faker.*;
```

## Benefits

- One faker library for ALL languages on Elide
- Consistent behavior across languages
