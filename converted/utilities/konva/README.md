# konva - Elide Polyglot Showcase

> **2D Canvas Framework for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- High-performance 2D graphics
- Event system
- Animation support
- **~2M downloads/week on npm**

## Quick Start

```typescript
import konva from './elide-konva.ts';

const rect = new Konva.Rect({
  x: 50, y: 50,
  width: 100, height: 100,
  fill: 'green'
});
console.log('Shape:', rect.width(), 'x', rect.height());
```
