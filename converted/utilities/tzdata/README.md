# tzdata - Elide Polyglot Showcase

> **Timezone database for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Comprehensive timezone data
- DST information
- **~3M downloads/week on npm**

## Quick Start

```typescript
import tzdata from './elide-tzdata.ts';

tzdata.getZone('America/New_York');
tzdata.getOffset('Europe/London');
tzdata.hasDST('Asia/Tokyo'); // false
tzdata.getAllZones();
```
