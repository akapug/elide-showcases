# GeoJSON Flatten - Elide Polyglot Showcase

> **One GeoJSON flattener for ALL languages** - TypeScript, Python, Ruby, and Java

Flatten MultiPoint, MultiLineString, MultiPolygon into individual features.

- Flatten multi-geometries
- Preserve properties
- **~30K+ downloads/week on npm**

## Quick Start

```typescript
import flatten from './elide-geojson-flatten.ts';

const flattened = flatten(multiPointFeature);
```
