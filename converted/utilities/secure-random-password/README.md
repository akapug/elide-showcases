# secure-random-password - Elide Polyglot Showcase

> **Secure password generation for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Cryptographically secure passwords
- Configurable length and character sets
- Number and symbol requirements
- **~500K downloads/week on npm**

## Quick Start

```typescript
import randomPassword from './elide-secure-random-password.ts';

randomPassword();                              // 16 chars with all types
randomPassword({ length: 24 });                // 24 chars
randomPassword({ symbols: false });            // No symbols
```
