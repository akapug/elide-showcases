# xxHash

xxHash algorithm in JavaScript for Elide (polyglot!)

Based on https://www.npmjs.com/package/xxhashjs (~100K+ downloads/week)

## Features

- xxHash algorithm in JavaScript
- Fast hashing algorithm
- Deterministic output
- Zero dependencies

## Quick Start

```typescript
import xxhashjs from './elide-xxhashjs.ts';
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript
import xxhashjs from './elide-xxhashjs.ts';
```

### Python (via Elide)
```python
from elide_xxhashjs import xxhashjs
```

### Ruby (via Elide)
```ruby
require 'elide/xxhashjs'
```

### Java (via Elide)
```java
import elide.xxhashjs.*;
```
