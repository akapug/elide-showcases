# ML-Matrix - Elide Polyglot Showcase

> **Matrix operations for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Matrix creation and manipulation
- Linear algebra operations
- Matrix decompositions
- Statistical functions
- **~500K downloads/week on npm**

## Quick Start

```typescript
import Matrix from './elide-ml-matrix.ts';

const m1 = new Matrix([[1, 2], [3, 4]]);
const m2 = new Matrix([[5, 6], [7, 8]]);

const product = m1.mmul(m2);
const transpose = m1.transpose();
const inverse = m1.inverse();
```
