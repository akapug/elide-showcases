# polynomial - Elide Polyglot Showcase

> **Polynomial operations for ALL languages** - TypeScript, Python, R, and Julia

## Features

- Polynomial evaluation
- Addition, multiplication
- Derivative
- **~20K downloads/week on npm**

## Quick Start

```typescript
import Polynomial from './elide-polynomial.ts';

const p = new Polynomial([1, 2, 3]); // 1 + 2x + 3x²
p.eval(2);
p.derivative();
```
