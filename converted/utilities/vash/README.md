# Vash - Elide Polyglot Showcase

> **One Razor-like engine for ALL languages** - TypeScript, Python, Ruby, and Java

Razor syntax for JavaScript templating, perfect for ASP.NET developers.

## Features

- Razor-like syntax
- JavaScript expressions
- HTML encoding
- **~500K downloads/week on npm**

## Quick Start

```typescript
import Vash from './elide-vash.ts';

const vash = new Vash();
const html = vash.render('<h1>Hello @name!</h1>', { name: 'World' });
```
