# dprint - Elide Polyglot Showcase

> **Pluggable formatter** - Fast, configurable code formatting

Pluggable and configurable code formatting platform.

## Features

- Extremely fast (Rust-based)
- Plugin system
- Multiple languages
- **~30K downloads/week on npm**

## Quick Start

```typescript
import dprint from './elide-dprint.ts';

const formatted = dprint.format('const\tx = 10;');
console.log(formatted);
```
