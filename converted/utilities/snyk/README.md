# snyk - Elide Polyglot Showcase

> **Vulnerability scanning for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Vulnerability scanning
- License compliance checking
- Dependency analysis
- Security monitoring
- **~5M downloads/week on npm**

## Quick Start

```typescript
import snyk from './elide-snyk.ts';

// Scan for vulnerabilities
const report = await snyk.test('.');

// Monitor continuously
await snyk.monitor('.');
```
