# tabbable - Elide Polyglot Showcase

> **Find tabbable elements for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Find tabbable elements
- Respect tabindex
- Handle disabled elements
- Shadow DOM support
- **~2M+ downloads/week on npm**

## Quick Start

```typescript
import { tabbable, isTabbable } from './elide-tabbable.ts';

const elements = tabbable(container);
const canTab = isTabbable(element);
```
