# Mithril - Elide Polyglot Showcase

> **One framework for ALL languages** - TypeScript, Python, Ruby, and Java

Modern client-side framework with fast rendering and small bundle size.

## Features

- Fast rendering
- Small bundle size
- Virtual DOM
- Component-based
- **~1M downloads/week on npm**

## Quick Start

```typescript
import Mithril from './elide-mithril.ts';

const vnode = Mithril.h('h1', null, 'Hello World');
const html = Mithril.render(vnode);
```
