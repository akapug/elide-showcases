# Random Item

Select random item from array for Elide (polyglot!)

Based on https://www.npmjs.com/package/random-item (~50K+ downloads/week)

## Features

- Select random item from array
- Easy to use API
- Type-safe
- Zero dependencies

## Quick Start

```typescript
import random_item from './elide-random-item.ts';

// Use the library
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript
import random_item from './elide-random-item.ts';
```

### Python (via Elide)
```python
from elide_random_item import random_item
```

### Ruby (via Elide)
```ruby
require 'elide/random_item'
```

### Java (via Elide)
```java
import elide.random_item.*;
```
