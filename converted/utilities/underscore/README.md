# Underscore for Elide

Functional programming helper library.

**Downloads**: ~7M/week on npm

## Quick Start

```typescript
import _ from './underscore.ts';

const doubled = _.map([1, 2, 3], x => x * 2);
const evens = _.filter([1, 2, 3, 4], x => x % 2 === 0);
```
