# UID Safe

URL and cookie safe UIDs for Elide (polyglot!)

Based on https://www.npmjs.com/package/uid-safe (~1M+ downloads/week)

## Features

- URL and cookie safe UIDs
- Unique and sortable
- Fast generation
- Zero dependencies

## Quick Start

```typescript
import uid_safe from './elide-uid-safe.ts';
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript
import uid_safe from './elide-uid-safe.ts';
```

### Python (via Elide)
```python
from elide_uid_safe import uid_safe
```

### Ruby (via Elide)
```ruby
require 'elide/uid_safe'
```

### Java (via Elide)
```java
import elide.uid_safe.*;
```
