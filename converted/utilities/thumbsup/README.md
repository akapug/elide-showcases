# Thumbsup - Elide Polyglot Showcase

> **One gallery generator for ALL languages** - TypeScript, Python, Ruby, and Java

Generate static video galleries with automatic thumbnails.

## Features

- Video gallery generation
- Automatic thumbnails
- Responsive layouts
- Album organization
- **~5K downloads/week on npm**

## Quick Start

```typescript
import build from './elide-thumbsup.ts';

await build({
  input: './videos',
  output: './gallery',
  title: 'My Gallery'
});
```
