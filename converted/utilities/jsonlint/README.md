# JSONLint - Elide Polyglot Showcase

> **JSON validator** - Parse and validate JSON

JSON parser and validator with detailed error messages.

## Features

- JSON syntax validation
- Detailed error messages
- Comments support
- **~100K downloads/week on npm**

## Quick Start

```typescript
import jsonlint from './elide-jsonlint.ts';

const result = jsonlint.validate('{"valid": true}');
console.log(result); // { valid: true, errors: [] }
```
