# compiled - Elide Polyglot Showcase

> **CSS-in-JS compiler - ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Build-time compilation
- Near-zero runtime
- Atomic CSS
- **~20K downloads/week on npm**

## Quick Start

```typescript
import { css, styled } from './elide-compiled.ts';

const className = css({ color: 'red' });
const Button = styled('button')({ background: 'blue' });
```
