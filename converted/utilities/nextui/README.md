# nextui - Elide Polyglot Showcase

> **Beautiful modern React UI for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Beautiful default theme
- Dark mode built-in
- Fully customizable
- **~50K downloads/week on npm**

## Quick Start

```typescript
import { Button, Card, Input } from './elide-nextui.ts';

Button.render({ color: 'primary', children: 'Click' });
Card.render({ isPressable: true, children: 'Content' });
```
