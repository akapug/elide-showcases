# focus-lock - Elide Polyglot Showcase

> **Focus locking for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Focus containment
- Auto focus first element
- Return focus on unlock
- **~300K+ downloads/week on npm**

## Quick Start

```typescript
import focusLock from './elide-focus-lock.ts';

const unlock = focusLock.lock('#modal');
// Later: unlock();
```
