# jest-extended - Elide Polyglot Showcase

> **One extended matcher library for ALL languages** - TypeScript, Python, Ruby, and Java

Additional Jest matchers for comprehensive testing.

## Features

- 100+ additional matchers
- Array matchers
- String matchers
- Number matchers
- Object matchers
- Zero dependencies
- **~500K downloads/week on npm**

## Quick Start

```typescript
import { matchers } from './elide-jest-extended.ts';

expect(matchers.toBeArray([1, 2, 3]).pass).toBeTruthy();
expect(matchers.toBeString('hello').pass).toBeTruthy();
```
