# Yocto Sleep - Tiny Sleep Function

Ultra-minimal sleep/delay utility in Elide.

Based on [yocto-sleep](https://www.npmjs.com/package/yocto-sleep) (~10K+ downloads/week)

## Quick Start

```typescript
import sleep from './elide-yocto-sleep';

await sleep(1000); // Sleep 1 second
console.log('Done!');
```
