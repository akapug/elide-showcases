# time-span - Elide Polyglot Showcase

> **Time span measurement for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Measure elapsed time
- Convert to various units
- **~500K downloads/week on npm**

## Quick Start

```typescript
import timeSpan from './elide-time-span.ts';

const span = timeSpan();
// ... do some work ...
const elapsed = span.end(); // milliseconds
span.seconds(); // in seconds
span.toString(); // human readable
```
