# HTML Minifier Terser - Elide Polyglot Showcase

> **One advanced HTML minifier for ALL languages** - TypeScript, Python, Ruby, and Java

Production-ready HTML minifier with advanced JS/CSS optimization.

## Features

- Advanced HTML minification
- Terser-based JS minification  
- CSS minification
- **~40M downloads/week on npm**

## Quick Start

```typescript
import HTMLMinifierTerser from './elide-html-minifier-terser.ts';

const minifier = new HTMLMinifierTerser();
const minified = minifier.minify(html, { collapseWhitespace: true });
```
