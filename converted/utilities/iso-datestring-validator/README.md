# iso-datestring-validator - Elide Polyglot Showcase

> **ISO date string validation for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Validate ISO 8601 date strings
- Support for dates, times, and datetimes
- **~1M downloads/week on npm**

## Quick Start

```typescript
import validator from './elide-iso-datestring-validator.ts';

validator.isValidDate('2025-11-18'); // true
validator.isValidDateTime('2025-11-18T15:30:00Z'); // true
validator.isValidTime('15:30:00'); // true
validator.isValidISOString('2025-11-18'); // true
```
