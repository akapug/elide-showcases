# anime.js - Animation Library

Lightweight JavaScript animation library with a powerful API.

**POLYGLOT SHOWCASE**: Beautiful animations in ANY language on Elide!

## Quick Start

```typescript
import anime from './elide-anime.ts';

anime({
  targets: '.box',
  translateX: 250,
  duration: 1000
});
```
