# react-dnd - Elide Polyglot Showcase

> **Drag and drop for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Full DOM control
- Flexible drag and drop
- **~500K downloads/week on npm**

## Quick Start

```typescript
import { DndProvider, useDrag, useDrop } from './elide-react-dnd.ts';

useDrag.create('card', { id: 1 });
useDrop.create('card');
```
