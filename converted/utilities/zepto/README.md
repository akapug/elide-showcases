# Zepto - Elide Polyglot Showcase

> **One minimalist JavaScript library for ALL languages** - TypeScript, Python, Ruby, and Java

A lightweight jQuery-compatible library for modern browsers.

## Features

- jQuery-compatible API
- Small footprint (~10KB)
- Mobile-focused
- Touch events support
- **~2M downloads/week on npm**

## Quick Start

```typescript
import Z from './elide-zepto.ts';

Z('div').addClass('active');
Z('.item').show();
Z('#name').val('Alice');
Z('button').on('tap', () => console.log('Tapped!'));
```
