# Screenshot Desktop - Elide Polyglot Showcase

> **One screenshot tool for ALL languages** - TypeScript, Python, Ruby, and Java

Capture screenshots of the desktop.

## Features

- Desktop screenshots
- Multi-monitor support
- Fast capture
- PNG/JPG format
- **~30K downloads/week on npm**

## Quick Start

```typescript
import screenshot from './elide-screenshot-desktop.ts';

const img = await screenshot({ screen: 0 });
// Save img buffer
```
