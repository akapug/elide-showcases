# mousetrap - Elide Polyglot Showcase

> **Keyboard shortcuts for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Keyboard shortcut binding
- Support for sequences
- Modifier keys
- Platform detection
- **~500K+ downloads/week on npm**

## Quick Start

```typescript
import mousetrap from './elide-mousetrap.ts';

mousetrap.bind('ctrl+s', () => console.log('Save!'));
mousetrap.bind('g i', () => console.log('Go to inbox'));
```
