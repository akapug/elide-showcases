# fecha - Elide Polyglot Showcase

> **Lightweight date formatting for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Simple date formatting and parsing
- Custom format masks
- **~8M downloads/week on npm**

## Quick Start

```typescript
import fecha from './elide-fecha.ts';

fecha.format(new Date(), 'YYYY-MM-DD'); // "2025-11-18"
fecha.format(new Date(), 'mediumDate'); // "Nov 18, 2025"
fecha.parse('2025-11-18', 'YYYY-MM-DD');
```
