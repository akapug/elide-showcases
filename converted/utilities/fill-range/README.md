## Features

- ✅ Works across TypeScript, Python, Ruby, and Java
- ✅ Zero dependencies
- ✅ High performance

## Quick Start

### TypeScript
```typescript
import module from './elide-fill-range.ts';
```

### Python
```python
from elide import require
module = require('./elide-fill-range.ts')
```

### Ruby
```ruby
module = Elide.require('./elide-fill-range.ts')
```

### Java
```java
Value module = context.eval("js", "require('./elide-fill-range.ts')");
```
