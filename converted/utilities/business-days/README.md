# business-days - Elide Polyglot Showcase

> **Business day calculations for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Calculate business days (excluding weekends)
- Holiday support
- **~500K downloads/week on npm**

## Quick Start

```typescript
import businessDays from './elide-business-days.ts';

businessDays.isBusinessDay(new Date());
businessDays.addBusinessDays(new Date(), 5);
businessDays.businessDaysDiff(startDate, endDate);
```
