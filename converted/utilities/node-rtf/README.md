# node-rtf - Elide Polyglot Showcase

> **RTF document generation for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Create RTF documents
- Text formatting
- Tables and lists
- **Polyglot**: Use from TypeScript, Python, Ruby, Java
- Zero dependencies

## Quick Start

```typescript
import { createDocument } from './elide-node-rtf.ts';

const doc = createDocument();
doc.addParagraph('Hello RTF!');
const rtf = doc.toString();
```
