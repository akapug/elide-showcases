# Markdownlint - Elide Polyglot Showcase

> **Markdown linter** - Style checker for Markdown files

A Node.js style checker and lint tool for Markdown files.

## Features

- 40+ Markdown rules
- Consistent style
- Auto-fixing
- **~500K downloads/week on npm**

## Quick Start

```typescript
import markdownlint from './elide-markdownlint.ts';

const result = markdownlint.validate('# Heading  \nSome text');
```
