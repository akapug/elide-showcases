# HyperID

High performance ID generator for Elide (polyglot!)

Based on https://www.npmjs.com/package/hyperid (~50K+ downloads/week)

## Features

- High performance ID generator
- Unique and sortable
- Fast generation
- Zero dependencies

## Quick Start

```typescript
import hyperid from './elide-hyperid.ts';
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript
import hyperid from './elide-hyperid.ts';
```

### Python (via Elide)
```python
from elide_hyperid import hyperid
```

### Ruby (via Elide)
```ruby
require 'elide/hyperid'
```

### Java (via Elide)
```java
import elide.hyperid.*;
```
