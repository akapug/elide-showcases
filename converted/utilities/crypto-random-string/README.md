# Crypto Random String

Generate cryptographically strong random strings for Elide (polyglot!)

Based on https://www.npmjs.com/package/crypto-random-string (~1M+ downloads/week)

## Features

- Generate cryptographically strong random strings
- Easy to use API
- Type-safe
- Zero dependencies

## Quick Start

```typescript
import crypto_random_string from './elide-crypto-random-string.ts';

// Use the library
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript
import crypto_random_string from './elide-crypto-random-string.ts';
```

### Python (via Elide)
```python
from elide_crypto_random_string import crypto_random_string
```

### Ruby (via Elide)
```ruby
require 'elide/crypto_random_string'
```

### Java (via Elide)
```java
import elide.crypto_random_string.*;
```
