# Underscore.string - Elide Polyglot Showcase

> **One comprehensive string library for ALL languages** - TypeScript, Python, Ruby, and Java

Complete string manipulation utilities inspired by underscore.string.

## Features

- capitalize, reverse, truncate
- count, words, slugify
- And many more utilities
- ~3M+ downloads/week on npm

## Quick Start

```typescript
import { capitalize, slugify, truncate } from './elide-underscore-string.ts';

capitalize("hello");                        // "Hello"
slugify("Hello World! Test");               // "hello-world-test"
truncate("hello world test", 10);           // "hello w..."
```
