# grommet - Elide Polyglot Showcase

> **Accessible React framework for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Responsive and accessible
- Themeable components
- **~50K downloads/week on npm**

## Quick Start

```typescript
import { Button, Box } from './elide-grommet.ts';

Button.render({ primary: true, label: 'Click' });
Box.render({ pad: 'medium', children: 'Content' });
```
