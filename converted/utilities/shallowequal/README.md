# ShallowEqual - Shallow Comparison Utility

Shallow comparison for React and general use on Elide.

## Quick Start

```typescript
import shallowequal from './elide-shallowequal.ts';

console.log(shallowequal({ a: 1 }, { a: 1 })); // true
```
