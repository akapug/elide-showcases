# Singularize - Elide Polyglot Showcase

> **One singularization library for ALL languages** - TypeScript, Python, Ruby, and Java

Convert plural English words to singular form with consistent behavior.

## Features

- Handles irregular words (people -> person)
- Uncountable words (sheep -> sheep)
- Pattern-based rules
- ~5M+ downloads/week on npm

## Quick Start

```typescript
import singularize from './elide-singularize.ts';
singularize("cats");      // "cat"
singularize("children");  // "child"
```
