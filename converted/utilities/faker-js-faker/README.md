# Faker.js

Modern faker library for Elide (polyglot!)

Based on https://www.npmjs.com/package/faker-js-faker (~2M+ downloads/week)

## Features

- Modern faker library
- Fast and efficient
- Type-safe
- Zero dependencies

## Quick Start

```typescript
import faker_js_faker from './elide-faker-js-faker.ts';
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript
import faker_js_faker from './elide-faker-js-faker.ts';
```

### Python (via Elide)
```python
from elide_faker_js_faker import faker_js_faker
```

### Ruby (via Elide)
```ruby
require 'elide/faker_js_faker'
```

### Java (via Elide)
```java
import elide.faker_js_faker.*;
```
