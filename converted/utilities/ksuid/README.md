# KSUID

K-Sortable Unique IDentifier for Elide (polyglot!)

Based on https://www.npmjs.com/package/ksuid (~50K+ downloads/week)

## Features

- K-Sortable Unique IDentifier
- Unique and sortable
- Fast generation
- Zero dependencies

## Quick Start

```typescript
import ksuid from './elide-ksuid.ts';
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript
import ksuid from './elide-ksuid.ts';
```

### Python (via Elide)
```python
from elide_ksuid import ksuid
```

### Ruby (via Elide)
```ruby
require 'elide/ksuid'
```

### Java (via Elide)
```java
import elide.ksuid.*;
```
