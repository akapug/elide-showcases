# Plates - Elide Polyglot Showcase

> **One PHP-style template engine for ALL languages** - TypeScript, Python, Ruby, and Java

Native PHP template system for JavaScript with familiar syntax.

## Features

- Native PHP-like syntax
- Template inheritance
- Sections
- **~200K downloads/week on npm**

## Quick Start

```typescript
import Plates from './elide-plates.ts';

const plates = new Plates();
const html = plates.render('Hello <?= $name ?>!', { name: 'World' });
```
