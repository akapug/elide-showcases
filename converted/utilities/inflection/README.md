# Inflection - Elide Polyglot Showcase

> **One inflection library for ALL languages** - TypeScript, Python, Ruby, and Java

Comprehensive string inflections for pluralize, singularize, ordinalize, and more.

## Features

- Pluralize and singularize
- Ordinalize numbers (1st, 2nd, 3rd)
- Dasherize and underscore
- Humanize strings
- Camelize
- ~8M+ downloads/week on npm

## Quick Start

```typescript
import { pluralize, singularize, ordinalize } from './elide-inflection.ts';

pluralize("cat");      // "cats"
singularize("boxes");  // "box"
ordinalize(1);         // "1st"
```
