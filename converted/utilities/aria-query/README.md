# aria-query - Elide Polyglot Showcase

> **ARIA specification utilities for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Complete ARIA role definitions
- ARIA attribute mappings
- DOM element role mapping
- Property validation
- **~10M+ downloads/week on npm**

## Quick Start

```typescript
import ariaQuery from './elide-aria-query.ts';

const role = ariaQuery.getRole('button');
const valid = ariaQuery.validatePropertyValue('aria-hidden', 'true');
```
