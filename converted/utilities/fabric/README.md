# fabric - Elide Polyglot Showcase

> **Powerful HTML5 Canvas Library for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Interactive canvas objects
- SVG support
- Rich text support
- **~3M downloads/week on npm**

## Quick Start

```typescript
import fabric from './elide-fabric.ts';

const rect = new fabric.Rect({
  width: 100,
  height: 100,
  fill: 'red'
});
console.log('Rectangle:', rect.width, 'x', rect.height);
```
