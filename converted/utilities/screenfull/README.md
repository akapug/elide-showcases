# screenfull - Elide Polyglot Showcase

> **Fullscreen API for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Cross-browser fullscreen
- Promise-based API
- Event handling
- **~300K+ downloads/week on npm**

## Quick Start

```typescript
import screenfull from './elide-screenfull.ts';

await screenfull.request();
await screenfull.exit();
```
