# React Native Camera - Elide Polyglot Showcase

> **One camera API for ALL languages** - TypeScript, Python, Ruby, and Java

Camera component for React Native.

## Features

- Photo capture
- Video recording
- Barcode scanning
- Face detection
- Flash control
- **~1M downloads/week on npm**

## Quick Start

```typescript
import { RNCamera } from './elide-react-native-camera.ts';

const camera = new RNCamera();
const photo = await camera.takePictureAsync();
```

## Documentation

elide run elide-react-native-camera.ts
