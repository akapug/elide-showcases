# Remark Lint - Elide Polyglot Showcase

> **Remark-based Markdown linting** - Pluggable Markdown linting

Plugin to lint Markdown with remark.

## Features

- Pluggable linting
- AST-based analysis
- Custom rules
- **~500K downloads/week on npm**

## Quick Start

```typescript
import remarkLint from './elide-remark-lint.ts';

const result = remarkLint.validate('# Heading\n# Heading');
```
