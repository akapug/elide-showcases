# sr-only - Elide Polyglot Showcase

> **Screen reader only utility for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- SR-only class generation
- Focusable variant
- CSS-in-JS support
- **~50K+ downloads/week on npm**

## Quick Start

```typescript
import srOnly from './elide-sr-only.ts';

srOnly.apply(element);
srOnly.applyFocusable(element);
```
