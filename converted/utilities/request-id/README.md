# request-id - Elide Polyglot Showcase

> **Request ID generation for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- UUID v4 generation
- Sequential IDs
- Timestamp-based IDs
- Custom formats
- **~20K downloads/week on npm**

## Quick Start

```typescript
import { RequestId } from './elide-request-id.ts';

const id = RequestId.generate();
const sequential = RequestId.sequential();
const timestamp = RequestId.timestamp();
```
