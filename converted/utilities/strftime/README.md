# strftime - Elide Polyglot Showcase

> **C-style date formatting for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- POSIX strftime formatting
- Locale support
- **~8M downloads/week on npm**

## Quick Start

```typescript
import strftime from './elide-strftime.ts';

strftime('%Y-%m-%d'); // "2025-11-18"
strftime('%A, %B %d'); // "Monday, November 18"
strftime('%I:%M %p'); // "03:45 PM"
```
