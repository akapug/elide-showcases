# React Native Splash Screen - Elide Polyglot Showcase

> **One splash screen library for ALL languages** - TypeScript, Python, Ruby, and Java

Splash screen for React Native apps.

## Features

- Show/hide splash screen
- Customizable duration
- Native implementation
- iOS and Android support
- **~300K downloads/week on npm**

## Quick Start

```typescript
import SplashScreen from './elide-react-native-splash-screen.ts';

SplashScreen.show();

// When ready
SplashScreen.hide();
```

## Documentation

Run the demo:

```bash
elide run elide-react-native-splash-screen.ts
```
