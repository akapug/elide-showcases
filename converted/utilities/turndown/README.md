# turndown - Elide Polyglot Showcase

> **HTML to Markdown conversion for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- HTML to Markdown conversion
- Custom rules
- Plugin system
- **Polyglot**: Use from TypeScript, Python, Ruby, Java
- Zero dependencies

## Quick Start

```typescript
import TurndownService from './elide-turndown.ts';

const turndown = new TurndownService();
const markdown = turndown.turndown('<h1>Hello</h1>');
console.log(markdown);
```
