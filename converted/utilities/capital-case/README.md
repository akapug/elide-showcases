# Capital Case - Elide Polyglot Showcase

> **One Capital Case library for ALL languages** - TypeScript, Python, Ruby, and Java

Convert strings to Capital Case (all words capitalized with spaces).

## Features

- All words capitalized
- Intelligent word splitting
- ~2M+ downloads/week on npm

## Quick Start

```typescript
import capitalCase from './elide-capital-case.ts';
capitalCase("helloWorld");  // "Hello World"
```
