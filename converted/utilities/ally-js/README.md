# ally.js - Elide Polyglot Showcase

> **Accessibility utilities for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Focus management
- Keyboard utilities
- Tab order management
- **~50K+ downloads/week on npm**

## Quick Start

```typescript
import ally from './elide-ally-js.ts';

ally.is.tabbable(element);
ally.query.focusable();
```
