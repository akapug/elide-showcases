# looks-same - Elide Polyglot Showcase

> **Image Comparison Tool for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Perceptual image comparison
- Ignore anti-aliasing
- Tolerance settings
- **~2M downloads/week on npm**

## Quick Start

```typescript
import lookssame from './elide-looks-same.ts';

looksSame('image1.png', 'image2.png', (error, equal) => {
  console.log('Images equal:', equal);
});
```
