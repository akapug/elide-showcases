# react-lazyload - Elide Polyglot Showcase

> **Lazy loading for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Lazy load components
- Image lazy loading
- **~300K downloads/week on npm**

## Quick Start

```typescript
import { LazyLoad, lazyLoadImage } from './elide-react-lazyload.ts';

LazyLoad.render({ height: 200, children: 'Content' });
lazyLoadImage('image.jpg');
```
