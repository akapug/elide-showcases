# Haversine Distance - Elide Polyglot Showcase

> **One distance calculator for ALL languages** - TypeScript, Python, Ruby, and Java

Simple haversine distance calculation.

## Quick Start

```typescript
import haversineDistance from './elide-haversine-distance.ts';

const dist = haversineDistance(
  { latitude: 40.7128, longitude: -74.0060 },
  { latitude: 51.5074, longitude: -0.1278 }
);
```
