# jasmine-core - Elide Polyglot Showcase

> **One BDD framework for ALL languages** - TypeScript, Python, Ruby, and Java

Official packaging of Jasmine's core files.

## Features

- BDD syntax
- Spies and mocks
- Async support
- Custom matchers
- Suite organization
- Zero dependencies
- **~3M downloads/week on npm**

## Quick Start

```typescript
import { describe, it, expect } from './elide-jasmine-core.ts';

describe('Math', () => {
  it('should add', () => {
    expect(1 + 1).toBe(2);
  });
});
```
