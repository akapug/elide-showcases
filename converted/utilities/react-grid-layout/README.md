# react-grid-layout - Elide Polyglot Showcase

> **Draggable grid layouts for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Draggable and resizable
- Responsive breakpoints
- **~200K downloads/week on npm**

## Quick Start

```typescript
import GridLayout from './elide-react-grid-layout.ts';

GridLayout.render({ layout: [{ i: 'a', x: 0, y: 0, w: 6, h: 2 }] });
```
