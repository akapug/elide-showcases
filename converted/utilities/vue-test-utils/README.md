# @vue/test-utils - Elide Polyglot Showcase

> **One Vue testing library for ALL languages** - TypeScript, Python, Ruby, and Java

Official testing utilities for Vue.js.

## Features

- Mount Vue components
- Query elements
- Trigger events
- Test props and emits
- Vue 3 support
- Zero dependencies
- **~500K downloads/week on npm**

## Quick Start

```typescript
import { mount } from './elide-vue-test-utils.ts';

const wrapper = mount(MyComponent);
expect(wrapper.text()).toContain('Hello');
```
