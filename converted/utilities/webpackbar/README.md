# webpackbar - Elide Polyglot Showcase

> **Progress bars for ALL build systems**

## Features

- Beautiful progress bar
- Build status
- Customizable colors
- **~500K+ downloads/week on npm**

## Quick Start

```typescript
import WebpackBar from './elide-webpackbar.ts';

const bar = new WebpackBar({ name: 'build' });
bar.setProgress(50, 'Building...');
bar.render();
```
