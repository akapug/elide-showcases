# snapshot-diff - Elide Polyglot Showcase

> **One snapshot diff library for ALL languages** - TypeScript, Python, Ruby, and Java

Diff snapshots for better testing.

## Features

- Diff two snapshots
- Readable output
- Works with Jest
- Zero dependencies
- **~100K downloads/week on npm**

## Quick Start

```typescript
import { snapshotDiff } from './elide-snapshot-diff.ts';

const diff = snapshotDiff({ name: 'Alice' }, { name: 'Bob' });
console.log(diff);
```
