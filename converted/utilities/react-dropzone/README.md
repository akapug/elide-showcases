# react-dropzone - Elide Polyglot Showcase

> **File upload component for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- HTML5 drag and drop
- File upload zone
- **~1M downloads/week on npm**

## Quick Start

```typescript
import { renderDropzone } from './elide-react-dropzone.ts';

renderDropzone({ accept: 'image/*', multiple: true });
```
