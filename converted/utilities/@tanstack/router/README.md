# TanStack Router - Elide Polyglot Showcase

> **One type-safe router for ALL languages** - TypeScript, Python, Ruby, and Java

Type-safe router for React applications with built-in data loading and caching.

## Features

- 100% type-safe routing
- Built-in data loading
- Route-based code splitting
- Search param validation
- Nested layouts
- Navigation guards
- Zero dependencies
- Polyglot: Use from TypeScript, Python, Ruby, and Java

## Quick Start

### TypeScript
```typescript
import { TanStackRouter } from './elide-tanstack-router.ts';

// Your code here
```

### Python
```python
from elide import require
router = require('./elide-tanstack-router.ts')

# Your code here
```

### Ruby
```ruby
router = Elide.require('./elide-tanstack-router.ts')

# Your code here
```

### Java
```java
Value router = context.eval("js", "require('./elide-tanstack-router.ts')");

// Your code here
```

## Why Polyglot?

- **Consistency**: Same behavior across all languages
- **Maintainability**: Single implementation to maintain
- **Type Safety**: Full TypeScript types available
- **Performance**: Native Elide performance
