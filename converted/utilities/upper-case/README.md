# Upper Case - Elide Polyglot Showcase

> **One UPPER CASE library for ALL languages** - TypeScript, Python, Ruby, and Java

Convert strings to UPPER CASE with intelligent word splitting.

## Features

- Splits words from camelCase, snake_case, kebab-case
- Converts to uppercase with spaces
- ~15M+ downloads/week on npm

## Quick Start

```typescript
import upperCase from './elide-upper-case.ts';
upperCase("helloWorld");  // "HELLO WORLD"
```
