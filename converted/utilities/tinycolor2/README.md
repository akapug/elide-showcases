# tinycolor2 - Elide Polyglot Showcase

> **Color manipulation - ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Color conversion
- Color manipulation
- Validation
- **~500K downloads/week on npm**

## Quick Start

```typescript
import tinycolor from './elide-tinycolor2.ts';

const color = tinycolor('#ff0000');
console.log(color.toHexString());
console.log(color.lighten(10).toRgbString());
```
