# Videostream - Elide Polyglot Showcase

> **One video streamer for ALL languages** - TypeScript, Python, Ruby, and Java

Stream video files over HTTP with range request support.

## Features

- HTTP video streaming
- Range request support
- Seek support
- Low memory usage
- **~10K downloads/week on npm**

## Quick Start

```typescript
import VideoStream from './elide-videostream.ts';

const stream = new VideoStream('video.mp4');
```
