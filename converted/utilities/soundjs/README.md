# soundjs - Elide Polyglot Showcase

> **Audio playback for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Simple audio playback
- Sound sprites
- Volume control
- **~20K+ downloads/week on npm**

## Quick Start

```typescript
import Sound from './elide-soundjs.ts';

Sound.registerSound('sound.mp3', 'sfx');
Sound.play('sfx', { volume: 0.5 });
```
