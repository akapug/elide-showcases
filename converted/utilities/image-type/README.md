# image-type - Elide Polyglot Showcase

> **Detect Image Type for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Detect image type from Buffer
- Support 20+ formats
- Fast and lightweight
- **~15M downloads/week on npm**

## Quick Start

```typescript
import imagetype from './elide-image-type.ts';

const type = imageType(buffer);
console.log('Type:', type?.ext, type?.mime);
```
