# React Native Image Picker - Elide Polyglot Showcase

> **One image picker for ALL languages** - TypeScript, Python, Ruby, and Java

Image picker for React Native.

## Features

- Photo library access
- Camera capture
- Image editing
- Video selection
- Multiple selection
- **~400K downloads/week on npm**

## Quick Start

```typescript
import { launchCamera, launchImageLibrary } from './elide-react-native-image-picker.ts';

const result = await launchCamera({ mediaType: 'photo' });
console.log(result);
```
