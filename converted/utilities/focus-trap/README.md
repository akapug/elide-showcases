# focus-trap - Elide Polyglot Showcase

> **Focus management for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Trap keyboard focus
- Handle tab navigation
- Escape key handling
- Return focus on deactivation
- **~500K+ downloads/week on npm**

## Quick Start

```typescript
import createFocusTrap from './elide-focus-trap.ts';

const trap = createFocusTrap('#modal');
trap.activate();
// Later: trap.deactivate();
```
