# node-pandoc - Elide Polyglot Showcase

> **Document conversion for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Convert between formats
- Markdown, HTML, PDF, DOCX
- LaTeX, EPUB, and more
- **Polyglot**: Use from TypeScript, Python, Ruby, Java
- Zero dependencies

## Quick Start

```typescript
import pandoc from './elide-node-pandoc.ts';

pandoc.convert('# Hello', 'markdown', 'html', (err, result) => {
  console.log(result);
});
```
