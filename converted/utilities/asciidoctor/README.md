# asciidoctor - Elide Polyglot Showcase

> **AsciiDoc processing for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- AsciiDoc to HTML
- DocBook conversion
- PDF generation
- **Polyglot**: Use from TypeScript, Python, Ruby, Java
- Zero dependencies

## Quick Start

```typescript
import Asciidoctor from './elide-asciidoctor.ts';

const asciidoc = '= Title\n\nContent';
const html = Asciidoctor.convert(asciidoc);
console.log(html);
```
