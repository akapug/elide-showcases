# react-virtualized - Elide Polyglot Showcase

> **Virtualization for large lists for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Efficient rendering of large lists
- Virtualized scrolling
- **~500K downloads/week on npm**

## Quick Start

```typescript
import { List } from './elide-react-virtualized.ts';

List.render({ rowCount: 1000, rowHeight: 35, rowRenderer: (i) => `Row ${i}` });
```
