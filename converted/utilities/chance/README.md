# Chance

Comprehensive random generator for Elide (polyglot!)

Based on https://www.npmjs.com/package/chance (~500K+ downloads/week)

## Features

- Comprehensive random generator
- Fast and efficient
- Type-safe
- Zero dependencies

## Quick Start

```typescript
import chance from './elide-chance.ts';
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript
import chance from './elide-chance.ts';
```

### Python (via Elide)
```python
from elide_chance import chance
```

### Ruby (via Elide)
```ruby
require 'elide/chance'
```

### Java (via Elide)
```java
import elide.chance.*;
```
