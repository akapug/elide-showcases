# Lower Case - Elide Polyglot Showcase

> **One lower case library for ALL languages** - TypeScript, Python, Ruby, and Java

Convert strings to lower case with intelligent word splitting.

## Features

- Splits words from camelCase, snake_case, kebab-case
- Converts to lowercase with spaces
- ~15M+ downloads/week on npm

## Quick Start

```typescript
import lowerCase from './elide-lower-case.ts';
lowerCase("helloWorld");  // "hello world"
```
