# React Native Haptic Feedback - Elide Polyglot Showcase

> **One haptic library for ALL languages** - TypeScript, Python, Ruby, and Java

Trigger haptic feedback on iOS and Android.

## Features

- Impact feedback
- Notification feedback
- Selection feedback
- Custom patterns
- iOS and Android support
- **~80K downloads/week on npm**

## Quick Start

```typescript
import ReactNativeHapticFeedback, { HapticFeedbackTypes } from './elide-react-native-haptic-feedback.ts';

ReactNativeHapticFeedback.trigger(HapticFeedbackTypes.impactMedium);
ReactNativeHapticFeedback.trigger(HapticFeedbackTypes.notificationSuccess);
```
