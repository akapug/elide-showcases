# jsbarcode - Elide Polyglot Showcase

> **Barcode Generator for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Generate barcodes
- Multiple formats (CODE128, EAN, UPC)
- SVG/Canvas output
- **~3M downloads/week on npm**

## Quick Start

```typescript
import jsbarcode from './elide-jsbarcode.ts';

JsBarcode.barcode('1234567890', {
  format: 'CODE128',
  width: 2,
  height: 100
});
console.log('Barcode generated');
```
