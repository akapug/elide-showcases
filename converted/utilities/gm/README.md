# gm - Elide Polyglot Showcase

> **GraphicsMagick/ImageMagick Wrapper for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Wrapper for GraphicsMagick and ImageMagick
- Powerful image manipulation
- Batch processing support
- **~3M downloads/week on npm**

## Quick Start

```typescript
import gm from './elide-gm.ts';

gm('input.jpg')
  .resize(800, 600)
  .write('output.jpg');
console.log('Image processed');
```
