# YAMLLint - Elide Polyglot Showcase

> **YAML linter** - Validate and lint YAML files

A linter for YAML files with indentation and formatting checks.

## Features

- YAML syntax validation
- Indentation checking
- Line length limits
- **~50K downloads/week on npm**

## Quick Start

```typescript
import yamllint from './elide-yamllint.ts';

const result = yamllint.validate('key: value  \n');
```
