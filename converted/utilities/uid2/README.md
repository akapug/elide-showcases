# UID2

UID generator v2 for Elide (polyglot!)

Based on https://www.npmjs.com/package/uid2 (~50K+ downloads/week)

## Features

- UID generator v2
- Unique and sortable
- Fast generation
- Zero dependencies

## Quick Start

```typescript
import uid2 from './elide-uid2.ts';
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript
import uid2 from './elide-uid2.ts';
```

### Python (via Elide)
```python
from elide_uid2 import uid2
```

### Ruby (via Elide)
```ruby
require 'elide/uid2'
```

### Java (via Elide)
```java
import elide.uid2.*;
```
