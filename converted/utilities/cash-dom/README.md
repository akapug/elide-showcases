# Cash-DOM - Elide Polyglot Showcase

> **One lightweight jQuery alternative for ALL languages** - TypeScript, Python, Ruby, and Java

A small, fast jQuery alternative for modern browsers.

## Features

- jQuery-like API
- Modern browser support
- Small bundle size (~6KB)
- Fast performance
- **~500K downloads/week on npm**

## Quick Start

```typescript
import cash from './elide-cash-dom.ts';

cash('div').addClass('active');
cash('.item').toggleClass('hidden');
cash('#name').val('Jane Doe');
cash('button').on('click', () => console.log('Clicked!'));
```
