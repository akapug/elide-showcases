# stats-lite - Elide Polyglot Showcase

> **Lightweight statistics for ALL languages** - TypeScript, Python, R, and Julia

## Features

- Mean, median, variance, stdev
- Percentiles
- Histogram
- **~30K downloads/week on npm**

## Quick Start

```typescript
import { mean, median, percentile } from './elide-stats-lite.ts';

const data = [1, 2, 3, 4, 5];
mean(data); // 3
percentile(data, 90);
```
