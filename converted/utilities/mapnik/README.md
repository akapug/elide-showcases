# Mapnik - Elide Polyglot Showcase

> **One map renderer for ALL languages** - TypeScript, Python, Ruby, and Java

High-quality map tile rendering engine.

## Quick Start

```typescript
import { Map } from './elide-mapnik.ts';

const map = new Map(256, 256);
const image = map.renderSync();
```
