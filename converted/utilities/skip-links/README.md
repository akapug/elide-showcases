# skip-links - Elide Polyglot Showcase

> **Skip navigation for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Skip to main content
- Skip to navigation
- Keyboard accessible
- **~20K+ downloads/week on npm**

## Quick Start

```typescript
import skipLinks from './elide-skip-links.ts';

skipLinks.add('Skip to main', 'main');
```
