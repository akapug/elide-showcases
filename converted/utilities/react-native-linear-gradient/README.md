# React Native Linear Gradient - Elide Polyglot Showcase

> **One gradient library for ALL languages** - TypeScript, Python, Ruby, and Java

Linear gradient component for React Native.

## Features

- Linear gradients
- Multiple colors
- Angle support
- Location control
- **~1M downloads/week on npm**

## Quick Start

```typescript
import { LinearGradient } from './elide-react-native-linear-gradient.ts';

const gradient = new LinearGradient({
  colors: ['#FF6B6B', '#4ECDC4'],
  start: { x: 0, y: 0 },
  end: { x: 1, y: 1 },
});
```
