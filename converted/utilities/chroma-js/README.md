# chroma-js - Elide Polyglot Showcase

> **Color library - ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Color space conversions
- Color scales
- Interpolation
- **~300K downloads/week on npm**

## Quick Start

```typescript
import chroma from './elide-chroma-js.ts';

const color = chroma('#ff0000');
console.log(color.hex());
console.log(color.rgb());
```
