# audio-loader - Elide Polyglot Showcase

> **Audio file loading for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Load audio files
- Decode audio data
- Promise-based API
- **~5K+ downloads/week on npm**

## Quick Start

```typescript
import load from './elide-audio-loader.ts';

const buffer = await load('sound.mp3');
console.log(buffer.duration);
```
