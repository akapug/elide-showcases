# Constant Case - Elide Polyglot Showcase

> **One CONSTANT_CASE library for ALL languages** - TypeScript, Python, Ruby, and Java

Convert strings to CONSTANT_CASE format for constants and environment variables.

## Features

- Uppercase with underscores
- Perfect for constants
- ~2M+ downloads/week on npm

## Quick Start

```typescript
import constantCase from './elide-constant-case.ts';
constantCase("helloWorld");  // "HELLO_WORLD"
```
