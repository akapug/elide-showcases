# dateformat - Elide Polyglot Showcase

> **Date formatting for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Powerful date formatting
- Named masks and custom formats
- **~15M downloads/week on npm**

## Quick Start

```typescript
import dateformat from './elide-dateformat.ts';

dateformat(new Date(), 'shortDate'); // "11/18/25"
dateformat(new Date(), 'isoDate'); // "2025-11-18"
dateformat(new Date(), 'dddd, mmmm d, yyyy'); // "Monday, November 18, 2025"
```
