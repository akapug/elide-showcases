# Prettier Plugin Organize Imports - Elide Polyglot Showcase

> **Organize imports** - Sort and organize imports automatically

Prettier plugin that makes imports sorted and organized.

## Features

- Auto-organize imports
- Remove unused imports
- Group by type
- **~100K downloads/week on npm**

## Quick Start

```typescript
import organizePlugin from './elide-prettier-plugin-organize-imports.ts';

const organized = organizePlugin.organize(code);
```
