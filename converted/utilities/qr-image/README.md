# qr-image - Elide Polyglot Showcase

> **QR Image Generator for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- QR code to image
- Stream output
- PNG/SVG support
- **~2M downloads/week on npm**

## Quick Start

```typescript
import qrimage from './elide-qr-image.ts';

const qr = qrImage.image('Hello', { type: 'png' });
console.log('QR image stream created');
```
