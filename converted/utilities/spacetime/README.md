# spacetime - Elide Polyglot Showcase

> **One timezone library for ALL languages** - TypeScript, Python, Ruby, and Java

Lightweight timezone library.

## Features

- Timezone conversion
- Date manipulation
- Zero dependencies
- **~100K downloads/week on npm**

## Quick Start

```typescript
import spacetime from './elide-spacetime.ts';

const s = spacetime();
console.log(s.format('iso'));
```
