# MurmurHash

Incremental MurmurHash3 for Elide (polyglot!)

Based on https://www.npmjs.com/package/imurmurhash (~10M+ downloads/week)

## Features

- Incremental MurmurHash3
- Fast hashing algorithm
- Deterministic output
- Zero dependencies

## Quick Start

```typescript
import imurmurhash from './elide-imurmurhash.ts';
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript
import imurmurhash from './elide-imurmurhash.ts';
```

### Python (via Elide)
```python
from elide_imurmurhash import imurmurhash
```

### Ruby (via Elide)
```ruby
require 'elide/imurmurhash'
```

### Java (via Elide)
```java
import elide.imurmurhash.*;
```
