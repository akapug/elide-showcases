# Flickity

Touch gallery for Elide (polyglot!)
Based on https://www.npmjs.com/package/flickity (~100K+ downloads/week)

## Features
- Touch gestures
- Responsive
- Wrap around
- Free scroll
- Autoplay
- Zero dependencies

## Quick Start
```typescript
import Flickity from './elide-flickity.ts';
const flickity = new Flickity(element, { wrapAround: true, autoPlay: 3000 });
```
