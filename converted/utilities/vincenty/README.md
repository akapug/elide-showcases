# Vincenty - Elide Polyglot Showcase

> **One Vincenty distance calculator for ALL languages** - TypeScript, Python, Ruby, and Java

High-precision distance calculation using Vincenty's formula.

## Quick Start

```typescript
import { distance } from './elide-vincenty.ts';

const dist = distance(40.7128, -74.0060, 51.5074, -0.1278);
console.log(dist); // meters
```
