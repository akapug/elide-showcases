# chrono-node - Elide Polyglot Showcase

> **Natural language date parsing for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Parse natural language dates ("tomorrow", "3 days ago", "next week")
- Relative date calculations
- **~3M downloads/week on npm**

## Quick Start

```typescript
import chrono from './elide-chrono-node.ts';

chrono.parseDate("tomorrow");
chrono.parseDate("3 days ago");
chrono.parseDate("in 1 week");
chrono.parse("Meet me tomorrow at 5pm");
```
