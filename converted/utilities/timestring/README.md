# timestring - Elide Polyglot Showcase

> **Parse time strings for ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- Convert time strings to seconds
- Support for multiple units
- **~500K downloads/week on npm**

## Quick Start

```typescript
import timestring from './elide-timestring.ts';

timestring('5 minutes'); // 300
timestring('1 hour', 'm'); // 60
timestring('2 days', 'h'); // 48
timestring('1h 30m'); // 5400
```
