# Geocoder - Elide Polyglot Showcase

> **One geocoder for ALL languages** - TypeScript, Python, Ruby, and Java

Geocode addresses to coordinates and reverse geocode.

## Quick Start

```typescript
import { geocode, reverse } from './elide-geocoder.ts';

const coords = await geocode('New York');
const address = await reverse(40.7128, -74.0060);
```
