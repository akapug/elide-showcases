# progress-bar-webpack-plugin - Elide Polyglot Showcase

> **Build progress for ALL build systems**

## Features

- Progress bar
- Time remaining
- Customizable format
- **~200K+ downloads/week on npm**

## Quick Start

```typescript
import ProgressBarWebpackPlugin from './elide-progress-bar-webpack-plugin.ts';

const progress = new ProgressBarWebpackPlugin();
progress.setProgress(50, 'Building...');
```
