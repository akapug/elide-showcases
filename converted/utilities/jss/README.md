# jss - Elide Polyglot Showcase

> **CSS-in-JS - ALL languages** - TypeScript, Python, Ruby, and Java

## Features

- JSON to CSS
- Plugin system
- Dynamic styles
- **~500K downloads/week on npm**

## Quick Start

```typescript
import { create } from './elide-jss.ts';

const jss = create();
const sheet = jss.createStyleSheet({
  button: { background: 'blue' }
});
```
