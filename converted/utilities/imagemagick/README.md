- Command-line ImageMagick wrapper
- Support for 200+ formats
- Advanced image effects
- **~2M downloads/week on npm**

## Quick Start

```typescript
import imagemagick from './elide-imagemagick.ts';

convert('input.jpg', [
  '-resize', '800x600',
  '-quality', '90'
], 'output.jpg');
```
