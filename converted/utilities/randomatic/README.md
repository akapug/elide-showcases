# Randomatic

Generate random strings with patterns for Elide (polyglot!)

Based on https://www.npmjs.com/package/randomatic (~200K+ downloads/week)

## Features

- Generate random strings with patterns
- Easy to use API
- Type-safe
- Zero dependencies

## Quick Start

```typescript
import randomatic from './elide-randomatic.ts';

// Use the library
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript
import randomatic from './elide-randomatic.ts';
```

### Python (via Elide)
```python
from elide_randomatic import randomatic
```

### Ruby (via Elide)
```ruby
require 'elide/randomatic'
```

### Java (via Elide)
```java
import elide.randomatic.*;
```
