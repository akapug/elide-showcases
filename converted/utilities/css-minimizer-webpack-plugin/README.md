# css-minimizer-webpack-plugin - Elide Polyglot Showcase

> **CSS optimization for ALL web frameworks** - TypeScript, Python, Ruby, and Java

## Features

- CSS minification
- Color and zero optimization
- Parallel processing
- **~3M+ downloads/week on npm**

## Quick Start

```typescript
import { minifyCSS } from './elide-css-minimizer-webpack-plugin.ts';

const minified = minifyCSS('.button { background: #ffffff; padding: 0px; }');
// Output: .button{background:#fff;padding:0}
```
