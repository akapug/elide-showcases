# FarmHash

Google FarmHash algorithm for Elide (polyglot!)

Based on https://www.npmjs.com/package/farmhash (~50K+ downloads/week)

## Features

- Google FarmHash algorithm
- Fast hashing algorithm
- Deterministic output
- Zero dependencies

## Quick Start

```typescript
import farmhash from './elide-farmhash.ts';
```

## Polyglot Examples

### JavaScript/TypeScript
```typescript
import farmhash from './elide-farmhash.ts';
```

### Python (via Elide)
```python
from elide_farmhash import farmhash
```

### Ruby (via Elide)
```ruby
require 'elide/farmhash'
```

### Java (via Elide)
```java
import elide.farmhash.*;
```

## Benefits

- One farmhash library for ALL languages on Elide
