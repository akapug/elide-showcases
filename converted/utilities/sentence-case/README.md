# Sentence Case - Elide Polyglot Showcase

> **One Sentence case library for ALL languages** - TypeScript, Python, Ruby, and Java

Convert strings to Sentence case format (first word capitalized).

## Features

- First word capitalized, rest lowercase
- Handles camelCase, snake_case, kebab-case
- ~2M+ downloads/week on npm

## Quick Start

```typescript
import sentenceCase from './elide-sentence-case.ts';
sentenceCase("helloWorld");  // "Hello world"
```
