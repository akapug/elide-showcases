# eigen - Elide Polyglot Showcase

> **Eigenvalue computation for ALL languages** - TypeScript, Python, R, and Julia

## Features

- Power iteration method
- Eigenvalue and eigenvector computation
- **~5K downloads/week on npm**

## Quick Start

```typescript
import { powerIteration } from './elide-eigen.ts';

const A = [[4, 1], [2, 3]];
const result = powerIteration(A);
console.log(result.eigenvalue);
```
