# Type Is

Type Is for Elide (polyglot!)

Based on https://www.npmjs.com/package/type-is (~10M+ downloads/week)

## Features

- Type checking
- Request inspection
- Multiple types
- Array support
- Zero dependencies

## Quick Start

```typescript
import type-is from './elide-type-is.ts';

// Basic usage
const result = type-is.main();
console.log(result);
```
