# Converted Showcases

4 complex npm packages demonstrating advanced Elide capabilities.

## Projects

### marked
**10M+ downloads/week** - Full-featured markdown parser

Complete implementation of CommonMark spec with extensions.

### validator
**9M+ downloads/week** - Comprehensive input validation library

Email, URL, credit card, date validation and more.

### decimal.js
**5M+ downloads/week** - Arbitrary precision decimal math

Eliminates floating-point errors in financial calculations.

### diff
Text diffing algorithm with multiple output formats.

## Why These Are "Showcases"

These packages are more complex than simple utilities:
- Large codebases (1000+ lines)
- Complex algorithms (AST parsing, validation rules, arbitrary precision math)
- Multiple features integrated together
- Production-ready implementations
