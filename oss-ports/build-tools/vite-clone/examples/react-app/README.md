## Configuration

See `vite.config.ts` for the Vite Clone configuration. This example uses:

- React plugin with Fast Refresh
- TypeScript support
- Path aliases
- CSS modules
- Development server with HMR

## Features Demonstrated

### Hot Module Replacement

Edit any React component and see changes instantly without losing state.

### TypeScript Integration

Full TypeScript support with type checking and IntelliSense.

### CSS Modules

Component-scoped CSS with automatic class name generation.

### Optimized Production Builds

Tree-shaking, code-splitting, and minification for optimal bundle sizes.

## Performance

Development server starts in <100ms and HMR updates in <50ms on average.
